Chromium Digest: 2026-07-13

Overview

Across a broad set of over 500 analyzed commits, including 406 highly relevant changes (ranging from commit (196beb2) to (872a849)), the day focused on architectural decoupling, build system modernization, and platform-specific stability improvements. No major breaking changes to stable public APIs were introduced.

Infrastructure & Build

  • The Siso build engine was upgraded to v1.5.23, adding thread-safe lazy loading and improved caching diagnostics. (fc39933) (f0c6b66)
    BuildInfra
  • Linux repository package generation is now deterministic using SOURCE_DATE_EPOCH to ensure byte-for-byte build output. (2cbb4a7)
    BuildLinux

Rendering & Graphics

  • ⚠️The legacy OpenGL-based ANGLE backend on macOS was removed to enforce a Metal-first rendering strategy. (75824ee)
    GPUmacOSCompositor
  • Enabled kDecodeScriptsInBlink by default to allow native script chunk decoding, reducing memory overhead and buffer copying. (9c33b2c)
    BlinkPerformanceV8

Security & Privacy

  • Enhanced Android Custom Tab security by restricting EXTRA_REMOTEVIEWS and secondary-toolbar updates to first-party callers, mitigating malicious data injection. (8a24769)
    AndroidSecurityPrivacy
  • Hardened Autofill by preventing the storage of Autofilled fields into the local Autocomplete database, ensuring sensitive data like addresses or payments are not persisted in search history. (e070f17)
    AutofillPrivacyHistory

Browser Architecture

  • Ongoing migration of Browser::profile() to BrowserWindowInterface::GetProfile() to decouple UI components from the legacy Browser class. (2342e76)
    BrowserProfilesUI
  • Re-architected AwSettings ownership in WebView to persist beyond individual instance destruction, resolving lifecycle-related configuration loss. (7cfc5f4)
    AndroidBrowser
