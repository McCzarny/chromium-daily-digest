Chromium Digest: 2026-05-10

Overview

This digest covers 11 commits from the Chromium project. Key highlights include the default enablement of the TreesInViz architecture, ongoing refactoring of the Android Custom Tabs toolbar, and various internal improvements to UI features, font rendering, and build configurations. No major breaking changes to stable public APIs were identified in this set of commits.

Compositor & Graphics

  • The TreesInViz feature, which moves more layer tree management into the Viz process, is now enabled by default on most platforms (excluding Linux). A subsequent fix was added to correctly propagate screenshot_destination_token in this mode, resolving timeouts in screenshot-based tests. CI bot configurations were also updated to reflect the new default. (b6929d0) (0099a3c) (aeba8a1)
    CompositorGPUInfra

Android

  • As part of an ongoing cleanup, the legacy implementation path for the Custom Chrome Tabs (CCT) toolbar has been removed. The modern, refactored toolbar implementation (CCTToolbarRefactor) is now used unconditionally, simplifying the codebase and removing obsolete metrics. (e6f2c6b)
    AndroidUI

UI & Features

  • A new feature parameter makes the tab glow hover delay configurable, allowing for UX experimentation without requiring code changes. (01b3460)
    UITabsPerformance
  • The TabContextualizationController is now accessed using the UserData-driven (UUD) pattern, a code health refactoring with no change in behavior. (78c3487)
    UITabs
  • A new metric, Glic.Onboarding.OptInImpression, has been added via the Glic mojo API to track impressions of a user onboarding screen. (aa9c8cb)
    UIPrivacy

Internals & Refactoring

  • In the font rendering engine, UNSAFE_TODO()s in OpenType CPAL parsing were eliminated by switching from std::unique_ptr<T[]> to the memory-safe base::HeapArray<T>. (d2bba9e)
    BlinkFontsSecurity
  • Build system configurations for protobuf were improved by adding a proper proto_library target for duration.proto, removing a build hack and ensuring correct path resolution. (f3144dd)
    BuildInfra
  • A test-only helper function, chrome::IsOffTheRecordBrowserInUse(), was inlined at its call sites as a preliminary step toward its eventual removal. (f141690)
    TestingBrowser
  • Automated commit to update translation files (.xtb) across various product components based on the latest source strings. (128654c)
    I18n
