Security & Code Hardening

  • Systematic migration of DCHECK assertions to fatal CHECK statements in content/browser/renderer_host to harden production builds against unexpected state. (e21a54a) (e8c46f0) (156705b) (b1b4324) (f54c069) (0af1503)

    SecurityPrivacy
  • Refactored MHTML generation to use Mojo Data Pipes, removing reliance on legacy sequence-bound pointers. (68855c5)
    SecurityBrowser

Web Platform & Rendering
