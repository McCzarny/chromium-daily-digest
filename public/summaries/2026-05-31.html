Chromium Digest: 2026-05-31

Overview

This digest analyzes 17 commits spanning internal refactoring, test stability, security hardening, and experimental feature enhancements. Key updates include transitioning prefetch callbacks to asynchronous execution, removing obsolete preloading HTTP headers, refining the experimental texElementImage2D WebGL API, and patching a potential browser-process crash vulnerability in the Bookmark Manager Private extension API. There are no major breaking changes affecting stable public APIs in this release.

Preloading & Web Networking

  • Transitioned PrefetchHandle callbacks (OnSuggestPrefetchCompletedOrFailed and OnPrefetchHeadReceived) from synchronous to asynchronous execution behind a kill switch flag to prevent synchronous state transition race conditions in the PrefetchContainerObserver lifecycle. (a163450)
    NetworkNavigation
  • Completed systematic retirement of the obsolete Purpose HTTP header from prefetch and prerender paths, removing its checking logic during prerender activation, removing it from the Network Context's cors_exempt_header_list parameters, and deleting the underlying purpose_header_ field in Blink's ResourceRequestHead. (c7d7e54) (098ef99) (de72e74)
    NetworkBlinkNavigation

Graphics, WebGL & Blink Rendering

  • Streamlined the non-standard, experimental texElementImage2D WebGL API (gated behind the CanvasDrawElement flag) by eliminating inapplicable parameters (level, format, type) and implementing strict validation parameters to restrict the internalformat argument to permitted formats. (dac54b9)
    WebGLBlinkGPU
  • Refactored compositor animation cancellation pipeline by removing the obsolete internal function CancelAnimationOnCompositor, allowing KeyframeEffect to directly interface with compositor layers without redundant active render surface checks. (f36e51b)
    CompositorBlink
  • Added comprehensive Web Platform Tests (WPT) to verify proper origin-tainting mechanisms are enforced on 2D and 3D canvas rendering contexts when utilizing cross-origin pattern assets or specific SVG filters. (dc4b9a2)
    BlinkHTMLTesting

Security & Extensions

  • Fixed a high-severity browser-process crash vulnerability in the bookmarkManagerPrivate.paste extension API by ensuring globally resolved bookmark node IDs are validated as direct children of the target destination parent node before fetching and utilizing their insertion indices. (5562e01)
    SecurityBookmarksExtensions

Infrastructure & Testing

  • Reverted Soft Navigation Core Web Vitals metric collection changes to resolve integration test failures in the SoftNavigationTest automated browser suite. (e38d144)
