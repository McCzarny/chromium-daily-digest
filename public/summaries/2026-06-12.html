Chromium Digest: 2026-06-12

Overview

A total of 627 commits were recorded on the Chromium main branch on 2026-06-12, with 492 identified as architecturally or feature-relevant for developers. Range: (6c47850) to (a0f367a).

Interface & API Evolution

  • ⚠️The bookmarks_api.mojom interface migrated from int64 identifiers to mojo_base.mojom.Uuid. This requires updates to all callers of the BookmarksService interface. (e96b0c4)
    BookmarksBrowser
  • ⚠️The direct-sockets-private permission policy has been removed. Isolated Web Apps must now use local-network and loopback-network granular policies. (f1b445b)
    SecurityNetwork

Media and Audio Infrastructure

  • Migrated platform/audio to the rustfft crate, enabling arbitrary FFT sizes and configurable render quantums while introducing new FFI dependencies. (14df232) (e9c0190)
    Media
  • Enabled H264 hardware encoding for Cast Streaming by default on Windows. (2a1143f)
    MediaWindows

Web Platform and Rendering

  • Applied overflow-clip-margin:content-box to <input type=image> to improve compliance with evolving HTML rendering specifications. (50d828d)
    BlinkHTMLCSS
  • Blocked untrusted DOMActivate events for <usermedia> elements, causing untrusted attempts to trigger InvalidStateError. (7994964)
    SecurityMedia

Infrastructure and Code Health

  • Runtime feature management now triggers a CHECK failure if mutable features are accessed before registration, enforcing stricter initialization. (8ab316f)
    BuildBase
  • Continued broad adoption of base::span for type-safe pointer handling in memory-mapped files and ref-counted interfaces. (4798477) (3be2af0)
    BaseSecurity
