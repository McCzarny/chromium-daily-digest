Week 14 of 2026 (2026-03-29 to 2026-04-03)

Chromium Weekly: 2026 Week 14

Overview

This week saw an exceptionally high volume of breaking changes across the Chromium codebase, impacting core browser APIs, Mojo interfaces, services, and extensions. Much of this was driven by the ongoing 'Bedrock' project to decouple UI from content and a general push for API modernization. Despite the churn, several key web platform features were enabled by default, including SharedWorker on all platforms and lazy loading for media elements. This summary covers 2449 relevant commits out of 3199 total across 6 days.

Widespread Breaking API Changes

  • ⚠️ BREAKING BREAKING CHANGE The "Bedrock" UI refactoring project continued its large-scale replacement of Browser* pointers with the new BrowserWindowInterface* in many public functions like chrome::OpenEmptyWindow and browser_commands, requiring significant updates for embedders. (6fdf39d) (4357394) (2174233)
  • ⚠️ BREAKING BREAKING CHANGE Numerous core C++ APIs were removed or altered. Notable changes include a new constructor for base::Thread, the removal of V8StringResource and ContentBrowserClient::GrantCookieAccessDueToHeuristic, and the renaming of blink::WebString::FromUTF16() to FromUtf16(). (7d91034) (ebf66e5) (ac62164) (cf3132f)
  • ⚠️ BREAKING BREAKING CHANGE Several stable Mojo interfaces were modified. The permission.mojom service now uses a PermissionStatusWithDetails struct, and the public video_conference.mojom Crosapi interface was deleted entirely. (36e8c6e) (645e372)
  • ⚠️ BREAKING BREAKING CHANGE The public IdentityManagerProvider class was removed, forcing consumers to use IdentityManagerFactory. Additionally, many public OAuth scope constants were removed from gaia_constants.h. (4db6a2e) (4fd8668) (9a12d39)
  • ⚠️ BREAKING BREAKING CHANGE Extension behavior on Android was aligned with desktop for chrome.tabs.create and chrome.tabs.move, and WebAuthn RP ID validation was made stricter across all platforms. (7214a10) (307d27c) (ecf43dd)
  • ⚠️ BREAKING BREAKING CHANGE A new NEW_SPLIT_VIEW value was added to the WindowOpenDisposition enum, which can cause compile-time failures in embedder code that uses switch statements without a default case. (7f076b4)

Web Platform & APIs

  • SharedWorker is now enabled by default on all platforms, including Android where it was previously in an Origin Trial, making the feature universally available. (6f4c113) (8bcc568)
  • The loading="lazy" attribute is now enabled by default for <video> and <audio> elements, deferring the loading of offscreen media to improve initial page load performance. (8744f9e)
  • CSS features saw advancements with name-only @container queries enabled by default, and experimental support added for background-clip: border-area to create effects like gradient borders. (b147722) (a6021f0)
  • ⚠️ BREAKING BREAKING CHANGE Content Security Policy (CSP) enforcement was tightened to treat <link rel=modulepreload> without an as attribute as a script (script-src-elem), which may break sites with misconfigured preloads. (ec9209f)

UI & Features

  • The experimental kWebUITabStrip feature has been disabled by default, effectively removing the WebUI-based tab strip. The New Tab Page 'Simplified UI' was also permanently removed. (880ccd9) (e54421d)
  • A mute button was added to the Video Picture-in-Picture window (behind a flag), and a keyboard shortcut (e.g., Alt+Shift+R) was added to open and close Reading Mode. (911883a) (bd8ce1f)
  • Tab management was improved with the addition of hover cards for tab group headers in the horizontal tab strip and API support for closing entire tab groups. (e8ce621) (7cd1de4)

Security

  • Security was hardened against compromised renderers by adding validation to prevent hijacking of mouse events, bypassing microphone permissions, and exploiting DevTools sessions via Fenced Frames. (b885cb0) (714840a) (a77ddcd)
  • File system access was restricted to mitigate vulnerabilities, blocking writes to .git/hooks directories via the File System Access API and fixing access bypasses in DevTools drag-and-drop. (fb916b2) (59d7444)
  • A Direct Sockets vulnerability allowing UDP multicast to bypass local network permissions was patched, and several Use-After-Free vulnerabilities were fixed across the codebase. (d3e49ac) (7b454b0) (daeb9ba)

Android

  • ⚠️ BREAKING BREAKING CHANGE The Android SDK was rolled to version 37.0, a major dependency update requiring embedders to update their build environments. (77027a5)
  • Performance was improved by enabling kMojoUseEventFd by default to reduce input delay and introducing a new off-main-thread prefetch flow (OMTPrefetch) for WebView. (04033a0) (64eed47)
  • Software decoding for 10-bit VP9 streams (Profile 2) was enabled for Android Desktop on arm64, and the cursor is now rendered during tab sharing. (201aca0) (ec66c01)
