Overview

Across 757 total commits, 581 were identified as relevant to desktop embedder development on Windows and macOS. The day focused on security hardening, infrastructure modularization, and performance optimizations. First commit: (6f312bb), last commit: (c416d0e).

Security & Network

  • Hardened MHTML document security by adding strict browser-side validation for CreateWebSocketConnector and CreateWebTransportConnector to prevent renderer-side bypasses. (7071428)
  • Restricted NetworkContext::CreateNetLogExporter to browser process usage to prevent unauthorized renderer access. (87c6908)
  • Improved HttpContentDisposition parser security by discarding comma-delimited tokens that could facilitate header manipulation vulnerabilities. (f195f95)

Browser UI & Architecture

  • Refactored Browser class window management into WindowMetadataController to improve modularity and support custom window UI elements. (fe731c2) (f3d320a)
  • Introduced OnPaintAsActiveChanged to the WebUI Mojo interface, enabling better CSS styling synchronization for components like the Tab Strip during window activation. (7696650)
  • Updated native desktop tab drag subsystem to trace screen-coordinate movements and manage interactive drag-and-drop bounds. (403db8f)

Core Utilities & Infrastructure

  • Updated base library i18n APIs by renaming LanguageCode to LanguageTag to align with BCP 47 semantics, including internal namespace migrations. (10cc07a)
  • Enhanced GPU client SharedImageExportResult to support multiple gpu::SyncToken containers for complex media pipelines like AR/XR and video capture. (45bb3da)
  • Expanded Mojo isolated connection API to allow custom MojoSendInvitationFlags, improving initialization permission control for multi-process architectures. (64fa90e)
