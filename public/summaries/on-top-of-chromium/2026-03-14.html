Chromium Digest: 2026-03-14

Overview

This digest covers 75 commits from the Chromium repository. After filtering for changes relevant to Windows and macOS embedders, focusing on browser APIs, UI, extensions, and security, 46 commits were analyzed. This summary highlights two major breaking changes in the UI and search components, alongside updates to security features, extension tooling, and internal refactoring. The changes span from (b1feb72) to (ea9f934).

Browser UI & Views

  • ⚠️ BREAKING BREAKING: The virtual method OnWidgetClosing has been removed from the public class WebUIBubbleDialogView. Embedders subclassing this view and overriding this method will experience build failures. The logic has been moved to a RegisterWindowClosingCallback pattern. (d18477d)
  • A new enum and helper class have been added to TabStripModel to allow for disabling close animations, specifically for tabs closed from the new 'Contextual tasks' feature. (592f4f6)
  • The chrome::CanSavePage() function has been refactored to operate over the BrowserWindowInterface, reducing dependencies in the Sharing Hub component. (3e8881c)
  • Miscellaneous UI and behavior fixes, including re-enabling tab movement shortcuts on macOS, adjusting the accessibility role of the Cast dialog's info icon, and various tweaks to the New Tab Page and vertical tabs. (d8e69d7) (773ba66) (0492460) (e9738c6) (6409484) (b4f6d75)

Browser Internals & Services

  • ⚠️ BREAKING BREAKING: To improve consistency, methods for confirming search engine removal have been renamed. TemplateURL::RequiresDeletionConfirmation is now RequiresRemovalConfirmation, and KeywordEditorController::ShouldConfirmDeletion is now ShouldConfirmRemoval. Embedders using these public APIs for search engine management must update their code. (ea9f934)
  • The PageContentAnnotationsService API has been extended to allow observation of on-device category classifier results, an internal enhancement for content analysis. (adbe66e)
  • Internal refactoring of the CriticalUserJourneyService to align with C++ style guidelines by replacing a unique_ptr with a direct object member. (3be785e)

Security

  • The garbage collection logic for Device-Bound Session Credentials (DBSC) has been moved to SessionStore to prevent accidental deletion of valid keys. The RestoreSessionBindingKey method signature was simplified by removing a priority parameter as part of this refactor for the experimental feature. (01c58d5)
  • Logic for the experimental FedCM API has been updated to allow navigation interception when there is an embedder login request, even if general interception is not enabled. (e1dfa92)
  • The migration to Enhanced Safe Browsing (ESB) will now be disabled if the Safe Browsing or JavaScript Optimizer preferences are managed by enterprise policy. (bfa6fc4)

Extensions

  • The extension API schema compiler now supports the nodoc attribute on individual enum values, allowing certain values to be excluded from generated documentation. This affects tooling and not runtime behavior. (f698dba)
  • A series of commits performed JavaScript modernization in various extension API tests, such as downloads, fileapi, and content_settings, replacing var with let/const and updating code style. These changes do not affect production APIs. (f8bc9ad) (928ae0c) (261f692) (5614bae) (c67313e) (01a06e8) (b091a6d) (05b9e19) (b1feb72) (6fb43a7)

PDF

  • Several internal refactorings and bug fixes were made to the experimental PDF Ink Signatures feature, including simplifying stroke ID recycling, fixing a command reference bug, and updating ID types. The feature flag's expiration was also extended. (3ad31ba) (54539cd) (396a915) (f2e85a3) (f38ba5e)

Infrastructure & Tooling

  • A new ESLint check has been added to disallow the use of chrome.send() in new Desktop WebUI code, enforcing the modern Mojo-based communication pattern. (d9c984b)
  • Significant refactoring of BUILD.gn files was undertaken to modularize chrome/browser/webapps and other components, improving build structure and dependency management. (c26595f) (f68a6c0) (ee1115b)
