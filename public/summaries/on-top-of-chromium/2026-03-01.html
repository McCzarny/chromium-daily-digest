Overview

This digest covers 3 selected commits out of an initial 10, focusing on changes relevant to browser UI, features, and core components, while excluding updates to the Blink engine, testing, and localization. Browser UI & Features

Core Components

This change formalizes an existing, albeit untracked, behavior to improve stability and prevent potential re-entrancy-related issues for embedders observing preference changes. (98cab97)
