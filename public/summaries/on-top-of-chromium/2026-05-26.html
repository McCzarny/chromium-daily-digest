Chromium Digest: 2026-05-26

Overview

This summary covers 715 total commits, with 539 identified as relevant to Windows and macOS desktop embedders. Significant activity included foundational shifts in extension infrastructure, networking stack security, and desktop UI modernization. Commits range from (33545f4) to (6efdfba).

Extension & Core API Infrastructure

  • ⚠️Manifest::Type was migrated from a legacy C-style enum to a strongly-typed enum class, requiring updates to any downstream code relying on implicit integer conversion or legacy naming styles. (82fbf20) (21dbf25)
    ExtensionsBase
  • Fixed a WebRequest API vulnerability related to Site Isolation, ensuring privileged extension processes are correctly authorized before registering listeners. (572cfa5)
    ExtensionsSecurity

Security & Network Architecture

  • RestrictedCookieManager access validation was consolidated to improve shared memory integrity, and Navigation parameters were sanitized to prevent sensitive metadata leakage. (99dfa68) (42ecaba)
    SecurityNetwork
  • Enhanced multi-networking support for QUIC and WebTransport by integrating target_network handles, and optimized DNS resolution by migrating to centralized job management. (533e55f) (4dca2fe)
    Network

Desktop UI & UX Refinements

  • Modernized the Windows system menu to use the Views-based framework and addressed a focus-stealing issue where PDF file associations opened in background processes. (9a1cbbd) (e3daa5b)
    WindowsUI
  • Introduced the DocumentPip widget delegate to modularize Picture-in-Picture functionality, separating it from the monolithic chrome/browser/ui target. (b7ed40c)
    UIBrowser

Build Infrastructure & Modernization

  • Switched default macOS build linking to the Apple linker, achieving significant build speed improvements (up to 6x) for developer and CI environments. (7842f92)
    BuildmacOS
  • Removed heavy update_client dependencies from the Windows elevation_service, reducing binary footprint through the use of base-level primitives. (d002ba7)
    BuildWindowsUpdater
