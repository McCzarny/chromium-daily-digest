Overview

This digest covers 641 total commits with 473 relevant to desktop embedders on Windows and macOS. The development cycle spanned from (a07fee6) to (a88d18b), focusing on architectural hardening, security protocol enforcement, and testing infrastructure modularization.

Public API & Infrastructure

  • ⚠️Removal of legacy ChildProcessSecurityPolicy methods (GrantFileForBrowserUpload, RevokeFileForBrowserUpload, CanReadFileForBrowserUpload) from the public interface, necessitating migration to ChildProcessSecurityPolicyImpl. (07d78ac)
    BrowserSecurityNavigation
  • Migration of renderer process identifiers in LoadURLParams and PrerenderAttributes from int to the content::ChildProcessId type to enhance type safety. (16c8f23) (940ee33)
    BrowserNavigationBase

Security & Networking

  • Enforced strict remote target port validation for P2P sockets and implemented header validation to restrict Connection headers to close and keep-alive tokens. (c4ef6a2) (4deb569)
    NetworkSecurity
  • Hardened Optimistic DNS by restricting it to TLS-secured connections and added automatic fresh DNS retries on TLS handshake failures. (93334f2) (a8d4244)
    NetworkSecurity
  • Simplified crypto::UnexportableKey hierarchy by merging interfaces into crypto::UnexportableSigningKey to streamline Device Bound Session Credentials (DBSC). (d6efd37)
    CryptoSecurity

Browser UI & Architecture

  • Refactored window-closing workflows into a centralized UnloadController and migrated singleton-dependent components to constructor-based dependency injection. (08cc46b) (23f150c) (d080b78)
    BrowserUI
  • Introduced support for delegating native child window layer management to the Views framework via SetLayerManagedByViews for better UI transformation support. (c82e07c)
    UICompositor
  • Improved Windows relaunch robustness with a parent-process-wait mechanism to prevent Profile in Use errors. (2aa54ca)
    BrowserWindows

Testing & Developer Utilities

    TestingBuildInfra
  • Replaced legacy UI event utilities with ui::test::EventGenerator in Aura-based browser tests to decouple from interactive test requirements. (18a0f9f)
    TestingUI
