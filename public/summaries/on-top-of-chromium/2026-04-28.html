Chromium Digest: 2026-04-28

Overview

This summary covers 553 relevant commits out of 697 total on the main branch for April 28, 2026. The day's changes include a large number of breaking API changes affecting core utilities, networking, UI, and permissions. Notable new features include a "Smart Restart" mechanism on macOS and UI improvements for PWAs and tab management. The first relevant commit was a1d0d57548d97d843a07e906f5a7d933ee29ed0a and the last was 41d95dc2830218c18337e0c4c87f5bc700d38ba3.

API & Core Libraries

  • ⚠️ BREAKING BREAKING: The browser shutdown function browser_shutdown::ShutdownPreThreadsStop() now returns a browser_shutdown::RestartMode enum instead of a bool to support new restart behaviors like the "Smart Restart" feature. (7002096)
  • ⚠️ BREAKING BREAKING: The base::LaunchProcess utility on Windows now opens the console handle with read-write access ("w+"). This alters how child processes detect a TTY and may break applications that relied on the previous write-only behavior. (6aaefd2)
  • ⚠️ BREAKING BREAKING: The ContentBrowserClient::FinishRtcDiagnosticLogging virtual method signature has changed, requiring embedders to add a new metadata parameter to their implementations. (05bd958)
  • ⚠️ BREAKING BREAKING: A new pure virtual method, OnPasswordFormsParsed, has been added to the password_manager::PasswordManagerInterface::Observer interface. All implementations of this observer must be updated. (c20c983)
  • ⚠️ BREAKING BREAKING: The media::VideoColorSpace class is now immutable. Direct member access (e.g., color_space.primaries) has been replaced with accessor methods (e.g., color_space.primaries()), breaking code that directly read or modified its state. (565307b)
  • ⚠️ BREAKING BREAKING: The public PushMessagingService::SubscriptionInfoCallback signature has been changed to include a new user_visible_only boolean parameter. Callback implementations must be updated. (23f0c1d)
  • ⚠️ BREAKING BREAKING: Core graphics interfaces now pass gfx::CALayerParams and gpu::SwapBuffersCompleteParams by move instead of by const reference. Custom implementations of viz::OutputSurfaceClient and ui::AcceleratedWidgetMacNSView will need to be updated. (750af5d)
  • ⚠️ BREAKING BREAKING: The media::MojoRendererService::Initialize method has been hardened to prevent re-initialization. A second call will now be treated as a bad message and terminate the renderer process. (2dfbd25)
  • ⚠️ BREAKING BREAKING: The signature for the public virtual method PdfInkModuleClient::DrawText() has changed to use an InkTextBoxAttributes struct, breaking existing implementations. (ee5efb8)
  • ⚠️ BREAKING BREAKING: Supervised user source files have been moved from chrome/browser/ to a new chrome/browser/supervised_user/ component. Build systems with hardcoded include paths will break. (f5f345d)
  • ⚠️ BREAKING BREAKING: The constructor for enterprise::connectors::FilesRequestHandler has changed due to a refactor. Code that directly instantiates this class must be updated. (18e969b)

Security & Permissions

  • ⚠️ BREAKING BREAKING: The core permissions model has been updated to allow a one-time (ephemeral) and a persistent permission grant to exist simultaneously for the same origin. Custom UI or logic assuming only one grant type can be active at a time will need to be updated. (3d9fd6e)
  • ⚠️ BREAKING BREAKING: The core utility base::DeletePathRecursively on Windows will no longer traverse reparse points (e.g., directory junctions). This is a security fix that prevents accidental deletion of content outside the intended directory, but it breaks any tools that relied on the previous, unsafe behavior. (aed056d)
  • ⚠️ BREAKING BREAKING: On Windows, ui::OpenFileViaShell now launches files using the explorer's token instead of inheriting the browser's. This is a significant behavioral change to prevent privilege escalation and will impact embedders relying on the previous token inheritance. (a68adec)
  • A vulnerability where a specially named file ('*') in a RAR archive could bypass Safe Browsing analysis has been fixed. (ac02d64)
  • Clipboard handling code on Windows was made more robust to prevent issues when processing malformed FILEGROUPDESCRIPTOR data. (6ad66e6)
  • A Use-After-Free (UAF) condition in NativeWidgetNSWindowBridge on macOS that could occur during fullscreen transitions has been fixed. (bc8da34)

Networking

  • ⚠️ BREAKING BREAKING: The net::URLRequest::Delegate interface has a new required pure virtual method, OnPlatformLocalNetworkAccessPermissionRequired. All embedders implementing this core networking delegate must add the new method. (0abb379)
  • ⚠️ BREAKING BREAKING: The core networking class net::HashValue is now immutable. A tag-only constructor has been removed, and the non-const span() method was also removed, breaking any code that constructed or modified a HashValue in place. (335a74c) (8823457)
  • The TCP socket pool limit for proxies, controlled by the kTcpSocketPoolProxyLimit feature, is now enabled by default, and the default limit has been increased from 32 to 128. (3564124)
  • A new navigation throttle has been added to attach the X-Geo HTTP header to Google-bound navigation requests on all platforms. (c8b8e95)

Extensions

  • ⚠️ BREAKING BREAKING: The autofillPrivate extensions API will now return an empty list instead of an error when its data managers are not yet loaded. This changes an error condition to a success condition, which can break logic in extensions that relied on the previous error-handling behavior. (b1e55a8)

Browser UI & Features

  • ⚠️ BREAKING BREAKING: The OmniboxAction struct within the public toolbar_ui_api_data_model.mojom interface has been refactored into a union. This is a breaking change for any client using this mojom API. (92e6fc8)
  • ⚠️ BREAKING BREAKING: The "card-linked offers" feature has been removed from Autofill. This includes changing the signature of the public function CreateCreditCardSuggestion, which will break any code calling it directly. (10d5999)
  • A new "Smart Restart" feature has been introduced on macOS to apply pending updates silently when no browser windows are open. A new public function chrome::AttemptRestartWithMode is available to trigger different restart types. (7002096)
  • The "Tabs From Other Devices" side panel now includes a search box to allow users to filter tabs by title or URL. (3186cd6)
  • The PWA installation dialog on Windows has been updated to include checkboxes for creating a desktop shortcut and pinning the app to the taskbar. (6f7528b)
  • When a video screen capture is aborted on macOS (e.g., via the native "Stop Sharing" button), any associated audio capture will now also be terminated. (fc933fa)
