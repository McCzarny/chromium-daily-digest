Breaking Changes

  • ⚠️ BREAKING The return types for base::SysInfo::AmountOfAvailablePhysicalMemory() and base::SysInfo::AmountOfVirtualMemory() have been changed from the deprecated ByteCount to base::ByteSize. Callers must update to handle the new return type. (c7e3442)
  • ⚠️ BREAKING Multiple public observer interfaces in //components/sync, including SyncManager::Observer and SyncServiceObserver, now inherit from base::CheckedObserver. Embedders implementing these interfaces must update their class definitions to inherit from the new base class. (3c26603) (702d4ff)
  • ⚠️ BREAKING The generic network error net::ERR_DNS_SERVER_FAILED has been removed and replaced by five more specific error codes (e.g., ERR_DNS_SERVER_FAILURE, ERR_DNS_REFUSED). Embedders must update error handling logic to check for the new set of errors. (0b88079)
  • ⚠️ BREAKING The public enum ExtensionUsageAction has been moved from ExtensionContextMenuModel to a new common header extensions/common/extension_usage.h. Code referencing ExtensionContextMenuModel::ExtensionUsageAction must be updated to use extensions::ExtensionUsageAction. (f4ac755)
  • ⚠️ BREAKING The pure virtual method RegisterWebSafeIsolatedScheme in the content::ChildProcessSecurityPolicy public interface has changed its signature by removing a boolean parameter. Embedders implementing or calling this method will have a build failure. (7f200a7)
  • ⚠️ BREAKING The public struct gfx::HdrMetadataAgtm has been removed. Code that directly accessed metadata.agtm on gfx::HDRMetadata must be updated to use the new accessor method metadata.getSerializedAgtm(). (932ca1c)
  • ⚠️ BREAKING The public preference keys kFingerprintingProtectionEnabled, kIpProtectionEnabled, and kTrackingProtectionLevel have been removed from components/privacy_sandbox/tracking_protection_prefs.h. Code reading or writing these prefs will no longer compile. (55cc993)
  • ⚠️ BREAKING The public method ContextualSearchContextController::CreateSearchUrl has been changed from synchronous (returning a GURL) to asynchronous (taking a callback), breaking its public contract. (d81d122)
  • ⚠️ BREAKING The public virtual method ShowEvaluationSurvey() has been removed from the page_info::MerchantTrustServiceDelegate interface as part of the removal of HaTS survey integration. (fbb2ad7)
  • ⚠️ BREAKING The public static helper method DataDecoder::ParseCborIsolated has been removed from services/data_decoder/public/cpp/data_decoder.h. Consumers must find an alternative for isolated CBOR parsing. (4a34e8f)
  • ⚠️ BREAKING The base::SyncMemoryPressureListenerRegistration class has been removed. Its functionality has been merged into base::MemoryPressureListenerRegistration, which is now the only class for this purpose. (b930199)
  • ⚠️ BREAKING The BufferPlane Mojo type has been removed from the public interface file ui/gfx/mojom/buffer_types.mojom. (d3967e3)
  • ⚠️ BREAKING The context_id field in the public struct lens::ContextualInputData has been changed from uint64_t to int64_t, which is a breaking change for consumers of this component's API. (9603e9b)
  • ⚠️ BREAKING A revert has removed the recently added Base64EncodeEarlyStartup() and Base64UrlEncodeEarlyStartup() functions from base/base64.h and base/base64url.h. (07dd31c)
  • ⚠️ BREAKING Public method signatures in the update_client component were changed to use base::OnceClosure instead of base::OnceCallback<void()>. (1ad280b)

Extensions

  • Support for structured cloning has been added to extension messaging APIs like runtime.sendMessage and tabs.sendMessage. This allows for richer data types (e.g., Map, Set, Date) to be passed in messages, augmenting the previous JSON-only serialization. (39b5b05)
  • A new error, CrxInstallErrorDetail::PROFILE_SHUTTING_DOWN, has been added. CrxInstaller will now abort installations if the target profile is shutting down, preventing potential crashes. (d07ba78)
  • A bug was fixed that could cause browser hangs when a webRequestBlocking listener was used in a service worker that was active but had not yet registered the listener. (022ce91)
  • Fixed a bug where extensions in "split" incognito mode could incorrectly override the New Tab Page. (c5bc723)

Networking & Security

  • The Local Network Access (LNA) feature now invalidates cached resources if the network configuration changes (e.g., connecting to a VPN), ensuring the browser re-fetches the resource to get the correct IP address and permission status. (a9b4054)
  • The Device Bound Session Credentials (DBSC) feature now notifies observers when a session is created via AddSession, allowing for better integration. (dee18f8)
  • The ForcedDiceMigration feature has been enabled by default, completing the migration to the DICE sign-in architecture. (ea4b1a4)
  • The lifetime of the Durable Message collector for DevTools is now tied to the DevTools Root Session, enabling network introspection of closed pages and browsing contexts. (55c4bf5)

Browser UI & Features

  • On Windows, the proactive warming up of dbghelp.dll in child processes has been removed to improve startup performance. The DLL will now be loaded on-demand. (b1ee8d0)
  • The dialog for Progressive Web App (PWA) updates has been updated to better support app migrations, with UI text and layout adjustments to make URL changes more noticeable. (60a30a1)
  • The Split View (Side-by-Side) feature is now unconditionally available, with its controlling feature flag removed. (1b9e069) (0fc0517)
  • On Windows, open/close animations for the Tab Search side panel were disabled to address jank. (bd356d0)
  • For macOS, new V-Sync provider classes have been added to support CADisplayLink in the browser process, altering the V-Sync flow for graphics. (b8f19a5)
  • On Windows, ASCII numerals will now be substituted for Arabic-Indic digits when the Arabic keyboard layout is active, improving input for that language. (aa8b65a)

Testing

  • Flaky tests on Windows were fixed by changing the wait condition from CompletedFirstVisuallyNonEmptyPaint to the more reliable RenderFrameSubmissionObserver::WaitForAnyFrameSubmission. (b135fd9)
  • The fake_dmserver test utility was ported to run on Linux and its unittests were moved to components_unittests for faster execution. (d46237b) (4b4e355) (ca89bf1)
  • The creation of WebApp objects in tests has been standardized to prepare for stricter construction constraints. (fcbed66)
