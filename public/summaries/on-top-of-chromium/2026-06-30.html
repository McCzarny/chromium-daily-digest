Overview

A total of 673 commits were processed, with 518 identified as relevant to Windows and macOS embedders. The analysis spans architectural refactorings, security hardening, and performance optimizations. The timeframe starts with commit (57ffb72) and concludes with (9babd84).

Security & Privacy

  • Fixed a WebView permission bypass where navigations ignored newwindow checks. Restrictive ServiceWorker permission checks for WebUSB implemented to prevent IPC abuse. (b27fe18) (3c1e3d4)
  • Hardened EyeDropper security by enforcing per-WebContents locks and applied scheme deny-lists to the bookmarkManagerPrivate API to prevent unauthorized internal navigation. (029ea4a) (b2b3eb4)

Infrastructure & Architecture

  • Expanded use of raw_ptr in STL containers for memory safety and migrated tracing logic to TRACE_STATE. DOM API relocated to the //extensions module to improve build modularity. (3702141) (c4e9a97) (986d096)
  • Introduced new Rust support for base::CommandLine and resolved procedural macro shadowing in mojo/rust to stabilize the IPC build chain. (d4c2008) (7ff173b)

UI & Browser Features

  • Enhanced macOS app shim stability monitoring and moved Settings pages toward the Lit framework. Added a new --unbounded-window-debug developer flag to visualize layout issues. (73aef89) (03e62dd) (8d5d421)
  • Unified Autofill policy service and enforced stricter data filtering. Removed legacy automated .user.js extension conversion logic. (96793e8) (38b617c)

Network & Performance

  • Improved address classification for IPv4-mapped IPv6 addresses and tightened UDP/QUIC socket security policies. (d81fbc0) (2b93b8b)
