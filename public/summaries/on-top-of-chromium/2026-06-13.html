Chromium Digest: 2026-06-13

Overview

A summary of recent Chromium updates encompassing 43 total commits. In accordance with guidelines, focus is restricted to APIs, extensions, networking, security, browser UI, and base utilities affecting Windows and macOS platforms, with webpage rendering (Blink/V8), mobile implementations, and ChromeOS specific changes excluded.

APIs & Extensions

  • Introduced the chrome.contextualTasksPrivate private extension API featuring getState and launchPanelInNewTab to support privileged component extensions. (61d3a3a)
  • Refactored extensions::Extension class internally to streamline manifest loading and clean up test-only utilities such as manifest version warning suppression. (1484d80)

Browser Security & Process Isolation

  • Migrated several ChildProcessSecurityPolicyImpl components to Rust, specifically handling dynamic Origin Agent Cluster (OAC) opt-in/opt-out registrations per BrowserContext. (5f9a8cf) (08b42ab) (406a72d)
  • Enhanced security validation in HLS playback by tracking the actual response origin of redirected manifests, preventing unauthorized cross-origin resource access. (eeef37a)

Performance & Base Utilities

  • Moved synchronous, blocking taskbar-pinning COM/RPC queries off the Windows UI thread to a background task runner to eliminate potential application hangs. (3a09680)
  • Optimized HistoryService by implementing QueryUrlIds(), a bulk querying interface that retrieves URL IDs for multiple GURLs in a single thread hop instead of using repeated single lookup operations. (0f1e22b)
  • Converted kSkiaGraphiteDawnUsePersistentCache into a standalone feature flag, enabling the browser process to check cache status before initializing the GPU process. (f5acb71)
  • Scoped DeclarativePerformanceObserver using DocumentUserData instead of WebContentsUserData to align metric tracker lifetimes with active documents and simplify back-forward cache lifecycle tracking. (0d22e4b)
  • Conducted a clean-up refactor replacing verbose base::OnceCallback<void(void)> and repeating equivalents with base::OnceClosure and base::RepeatingClosure aliases. (edbab6a)
  • Encapsulated ReadKey helper inside HlsNetworkAccessImpl to restrict its visibility to the internal HLS network layer. (a29e5d8)

Browser UI & Window Management

  • Refactored the Windows updater's dialog styling to scale corner radii dynamically and support system high contrast borders, using off-screen double-buffering to eliminate flickering. (97a7f3e)
  • Disabled the kZeroCopyDesktopCapture feature flag globally on Windows and Linux to serve as a killswitch for a critical desktop capturing regression. (01106d5)
  • Refactored internal side panel UI logic, improving WebContents discard on panel close, addressing composebox layout focus issues, and updating realbox AI-mode query animations. (411acd6) (ebbaf81) (a6d8e53) (5b06c6d)
