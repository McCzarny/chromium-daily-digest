Chromium Digest: 2026-06-24

Overview

Across 692 total commits on the main branch, 493 were identified as relevant to Windows and macOS embedders, focusing on browser architecture, security, and UI framework modernization. Changes span from the first commit (52ae797) to the last commit (a9e7d75).

Extensions & Web Apps

  • Introduced kWebRequestPerContextEventDispatch to enable per-renderer context event dispatching, moving away from sub-event listener patterns. WebRequest constants have been refactored into a shared internal header for better cross-process validation. (c6b2d17) (1747ca5)
    ExtensionsNetwork
  • Isolated Web Apps (IWA) are now prohibited from using chrome.tabs.duplicate; developers should switch to chrome.windows.create to maintain security boundaries. IwaInstaller API was simplified to include Profile* dependencies. (fb54f5d) (d478d49)
    ExtensionsBrowser

Browser Security & Privacy

  • Credential autofill logic hardened: PasswordManagerClient now uses url::Origin for filling checks instead of GURL to prevent sandboxed iframe exploitation. FedCM and URL rewrite rules also received security hardening to block same-origin bypasses. (94f0fa3) (3bf03fd) (2af692f)
    SecurityPasswordManagerPrivacy
  • Initiated SHA-1 deprecation campaign across crypto/ modules. Migrated internal variations and network-level encryption logic to the crypto/hpke library for improved maintenance. (61e3eac) (c1f15fa)
    CryptoSecurity

Browser Architecture & UI

  • Accelerated decoupling of the Browser class by transitioning direct window management access to BrowserWindow::FromBrowser(). Internal UI elements are being migrated to the declarative Actions Framework. (652e373) (4a28b82)
    BrowserUI
  • Implemented native acrylic/translucent 'glass' effects for desktop browser toolbars. Task Manager UI removed legacy layout flags following design stabilization. (ea54089) (8ad88f2)
    UIWindowsmacOS

Developer Tools & Infrastructure

  • DevTools overrides (Headers, UserAgent, Media) now correctly apply to WebSocket handshake requests. AppleScript execution of arbitrary JS is now gated behind DevTools permissions on macOS. (5459d3d) (f66828a)
    DevToolsmacOS
  • Test suite modularization continues, moving monolithic test targets into isolated source sets for improved build times. MiraclePtr has been expanded to cover more raw pointers within templated containers. (cb1c26a) (5bcf476)
    TestingBuildBase
