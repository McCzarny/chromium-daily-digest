Summary for 2025-11-24 on branch 'main'

Chromium Digest: 2025-11-24

Overview

This summary covers 444 commits on the main branch, with 303 identified as relevant to developers building on Chromium for Windows and macOS. The day saw a significant number of breaking changes to core interfaces, networking classes, and browser preferences, requiring action from embedders. Notable feature updates include enabling a new tabbed layout by default and introducing a new enterprise policy for AI content sharing. The first relevant commit was (65fd1d1) and the last was (838bb0e).

Breaking Changes

  • ⚠️ BREAKING A new pure virtual method, GetOAuthConsumerFromId, was added to the SigninClient interface. Embedders with custom SigninClient implementations must add an implementation for this method to avoid build failures. (5657390)
  • ⚠️ BREAKING The NetworkTimeObserver class has been refactored. The constructor now requires a NetworkTimeTracker* and the class manages its own subscription lifecycle. Manual calls to AddObserver/RemoveObserver are no longer correct and must be removed. (4a5a3bc)
  • ⚠️ BREAKING A large number of deprecated browser preferences, scheduled for removal in or before November 2024, have been deleted from browser_prefs.cc. Embedders or extensions relying on these legacy preferences will experience behavioral changes or breakage. (980ce9d)
  • ⚠️ BREAKING Transitive includes for ui/gfx/geometry/rect.h and rect_f.h have been removed from core public headers like content/public/browser/web_contents.h. Code that relied on these headers to implicitly include gfx::Rect or gfx::RectF will need to add the includes directly. (5ac7f34)
  • ⚠️ BREAKING The ConnectorUploadRequest class has been moved from chrome/browser/safe_browsing/cloud_content_scanning/ to components/enterprise/connectors/core/cloud_content_scanning/ and is now in the enterprise_connectors namespace. Includes and namespace references must be updated. (8550f1e)
  • ⚠️ BREAKING The callback signature for SimpleURLLoader::DownloadToStringOfUnboundedSizeUntilCrashAndDie has changed to use std::optional instead of std::unique_ptr. Callers must update their callback handlers. (460bab4)
  • ⚠️ BREAKING The public helper class PlusAddressSuggestionHelper has been removed and its logic inlined into PlusAddressServiceImpl. Direct usage of this class will no longer compile. (6346844)
  • ⚠️ BREAKING The deprecated sigslot::has_slots<> base class has been removed from public classes, including remoting::protocol::IceTransportChannel. This will break code that inherits from these classes or uses the old sigslot mechanism. (ec77c82)

Security & Enterprise

  • A new "approval mode" has been introduced for the DevTools remote debugging server, controlled by the kDevToolsRemoteDebuggingEnabled preference and the kDevToolsRemoteDebuggingAllowed enterprise policy. The server can now be toggled via chrome://inspect. (2e576ad) (2748580)
  • A new enterprise policy, SearchContentSharingSettings, was added to allow administrators to control content sharing with AI Mode, independent of existing AI settings. (6e7bf35)
  • Several Third-Party Cookie Deprecation (TPCD) related origin trials (Tpcd, TopLevelTpcd, LimitThirdPartyCookies) and their associated test utilities have been removed from the codebase as they are no longer active. (03eb3e4)
  • The Connection-Allowlist feature has been updated to use string patterns instead of origins, allowing for more robust matching syntax via SimpleUrlPatternMatcher. (d6a4b93)

Browser UI & Features

  • The new tabbed browser layout, which supports the "Toolbar Height Side Panel" (THSB), is now enabled by default. It can still be controlled via a chrome://flags entry. (76dd43c)
  • The "Your Saved Info" settings page has been updated. The route has been changed from /addresses to /contactInfo (with a redirect), and a new "Contact info" menu item has been added to the main app menu. (0634061) (57b38f3)
  • A new entry point for the Vertical Tabs feature has been added to the chrome://settings/appearance page. (72b5104)
  • A new feature flag, kAccessibilitySequentialFocus, has been introduced to control an experimental reactive focus synchronization behavior for accessibility features. (0cd861c)

Extensions

  • A new private API, metricsPrivate.recordExtensionUsageUkm, was added to allow specific allowlisted extensions to record UKM events. (e9a99b5)

Core & Base Library

  • A new overload for base::CommandLine::Init() was added to accept a std::vector, providing a safer, more modern way to initialize the command line from a vector of arguments. (31416c4)

Testing & Tooling

  • The HistogramTester was updated to ensure PartitionAlloc-everywhere (PUMA) histograms are included in tests by default, improving memory-related test coverage. (3be4348)
  • A new debugging screen for Variations SeedFiles has been added to the chrome://metrics-internals page, aiding in the diagnosis of field trial configuration issues. (72f0a6b)
