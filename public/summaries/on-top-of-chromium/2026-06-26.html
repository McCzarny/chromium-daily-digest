Chromium Digest: 2026-06-26

Overview

This digest covers 725 total commits for the Chromium main branch on 2026-06-26, with 534 commits identified as relevant to Windows and macOS embedders. The development focus for the day included major API refactoring for WebSocket handling, enterprise-grade security hardening, and significant performance optimizations across networking and storage subsystems. The range extends from first commit (45cb8c0) to last commit (1142b53).

API & Interface Changes

  • ⚠️The ContentBrowserClient public API for WebSocket management has undergone a major signature change. Methods GetWebSocketOptions and CreateWebSocket now utilize a nested WebSocketOptions struct instead of legacy uint32_t parameters. Embedders must update overridden implementations to avoid compilation errors. (7452e34)
    BrowserNetwork

Security & Enterprise

  • Enhanced extension security by transitioning ExtensionActionRunner to use GetLastCommittedURL, mitigating potential TOCTOU exploits. Additionally, introduced strict enterprise policies for ProcessIsolationEnabled on Windows and integrated new AutofillSettings policies. (c5a7017) (dc3bed1) (48838b5)
    ExtensionsSecurityEnterprise
  • Hardened site-isolated storage by migrating SessionStorageLevelDB to atomic metadata writing, preventing cross-site data leakage from orphaned storage map files. (c7bc45a)
    StorageSecurity

Performance & Architecture

  • Optimized database performance by removing sql::DatabaseOptions preloading, resulting in a significant reduction in macOS lock-acquisition faults. Parallelized DNS resolution in TcpConnectJob to reduce overall connection latency. (3f9807e) (b4b9726)
    PerformanceNetworkmacOS
  • Refactored BrowserView and Browser window lifecycle management to move initializations into InitPostWindowConstruction, supporting the long-term Project Bedrock architectural migration. (aae2e42)
    BrowserUI

Testing & Build Infrastructure

  • Increased turboci pipeline allocation to 100% on primary desktop trybots and supported remote execution via Siso/Reclient for TypeScript library pipelines to reduce developer build cycles. (e092369) (191a5ac)
    BuildTesting
