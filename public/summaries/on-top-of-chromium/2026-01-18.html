Overview

This digest covers 8 commits, with analysis focusing on 2 changes relevant to browser-level APIs, extensions, and core utilities after filtering out platform-specific, rendering, and localization updates. Base Library

  • ⚠️ BREAKING The utility function base::Contains and its corresponding header base/containers/contains.h have been removed from the codebase. The commit message indicates all internal callers were migrated. Projects built on top of Chromium that used this public helper function will need to be updated to use alternatives, such as a container's native .contains() method or std::find. (4e6249c) (73901c6)

Extensions

  • InstallStageTracker, a class for monitoring extension installation progress, has been refactored out of //chrome and moved into the //extensions component. While no behavioral changes are intended, this file relocation may require downstream projects to update their build configurations and include paths. (7bdf603)
