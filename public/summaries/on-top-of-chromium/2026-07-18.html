Chromium Digest: 2026-07-18

Overview

An analysis of 41 commits was completed, prioritizing updates relevant to downstream Windows and macOS Chromium-based browser deployments. This review, covering commits (c13a086) through (1340451), confirmed that no major breaking changes were made to stable public APIs.

Security & Network

  • Introduced a strict uncertainty threshold (kMaxUncertaintyForNetworkTime = 45s) within CertVerifyServiceFactory to reject highly uncertain network times and secure SSL/TLS certificate verification. (3b4225c)

Browser UI & Desktop Features

  • Refactored contextual task tracking to selectively clear underlines only on the active, navigating tab. Registered Glic explicit ThemeSources, optimized Composebox token management, and added the composebox-skills flag. (4a3ce88) (93ef1e4) (f8a1e58) (1078e8a) (c13a086)
  • Optimized autofill suggestion layouts by hiding suggestion popups inside AttemptToDisplayAutofillSuggestions() to prevent display of stale data. (9e6a4c5)
  • Rolled out fieldtrial configurations for the DesktopNtpDriveModuleLink on desktop platforms. (1340451)

Base, Core APIs & Services

  • Reverted Persistent Memory Allocator metrics logging because writing to processing processes caused a spike in Chrome Installer process crashes on Windows. (ed40775)
  • Addressed UNSAFE_TODO raw pointer concerns within the services/device/usb/ descriptor parsing libraries to boost device security. (07abc56)
  • Cleaned up Chrome Remote Desktop signaling code by removing legacy XML parsing routes and removing the obsolete VideoStream::Observer interface. (7f3e4f7) (2ff00bf)
