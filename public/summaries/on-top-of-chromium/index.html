Chromium Digest: 2025-11-25

Overview

This summary covers 525 relevant commits out of 675 total, from commit (4447638) to (4917237). The day was marked by numerous breaking changes across core APIs, networking, and extensions, alongside the complete removal of the Tab Scrolling feature. Developers should pay close attention to API signature changes in Mojo, networking, and extension creation, as well as widespread build breaks due to header cleanup.
