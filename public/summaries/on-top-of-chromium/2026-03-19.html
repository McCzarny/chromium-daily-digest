Summary for 2026-03-19 on branch 'main'

Chromium Digest: 2026-03-19

Overview

This summary covers 370 relevant commits out of 514 total for the main branch on March 19, 2026. The day was marked by a large number of breaking changes across core APIs, browser features, and developer tools, requiring significant attention from embedders. Key updates include modifications to content::WebContents, the removal of the Tab Organization service, and an upgrade of the core ICU library. The first relevant commit was (37f1b6e) and the last was (b26b738).

Core APIs & Interfaces

  • ⚠️ BREAKING BREAKING CHANGE: The content::WebContents interface has a new pure virtual method, RequestSelectionBounds. All embedders subclassing content::WebContents must implement this new method to avoid build failures. (8020df7)
  • ⚠️ BREAKING BREAKING CHANGE: The core International Components for Unicode (ICU) library has been upgraded to version 78.2. This can cause significant behavioral changes in internationalization APIs, such as URL formatting and IDNA spoof checking, potentially breaking downstream parsers that expect older formats (e.g., timezone formatting changing from "GMT" to "GMT+0"). (f72d48e)
  • ⚠️ BREAKING BREAKING CHANGE: The content::ContentBrowserClient interface has a new virtual method, ShouldDenySharedWorkerExtendedLifetime. While it has a default implementation, embedders may need to override it to correctly handle the new SharedWorkerExtendedLifetimeEnabled enterprise policy. (d53e73e)
  • ⚠️ BREAKING BREAKING CHANGE: The signature for BrowserUserEducationInterface::CanShowFeaturePromo() has been changed to include a PassKey parameter, breaking all existing calls to this public method. (48c0c42)
  • ⚠️ BREAKING BREAKING CHANGE: The public method WebView::SetCrashedOverlayView in ui/views/controls/webview/webview.h has been re-introduced after a recent removal. Code that was updated to accommodate its removal will now be broken. (ccf079f)
  • ⚠️ BREAKING BREAKING CHANGE: Two public methods, CanCopySharedImageToGLTextureViaSkia and CanCopySharedImageToGLTextureViaTextureCopy, have been removed from the GLES2Interface. Embedders directly calling these low-level GPU interface methods will need to find alternative implementations. (eeb6937) (5a7ae68)

Browser UI & Features

  • ⚠️ BREAKING BREAKING CHANGE: The Tab Organization feature has been removed. This includes the TabOrganizationService, its factory, and the associated IDC_ORGANIZE_TABS command. Projects that integrated with this service will break and must remove their dependencies. (4a44e72)
  • ⚠️ BREAKING BREAKING CHANGE: The API for showing context menus from WebUI toolbar buttons has been updated. The showContextMenu method in toolbar_ui_api.mojom now requires a gfx.mojom.Rect (screen_rect) instead of a gfx.mojom.Point. WebUI implementations must be updated to pass the full bounding rectangle of the anchor element. (4c3a21b)
  • On macOS, the First Run dialog has been fully migrated from a native Cocoa implementation to the cross-platform Views framework. The public-facing ShowFirstRunDialog() function remains, but its implementation is now exclusively Views-based. (5ce0c2b)

Web APIs

  • ⚠️ BREAKING BREAKING CHANGE: The error handling of the Digital Credentials API has been changed. When a user cancels a request, the API now rejects the promise with a NotAllowedError instead of a NetworkError. Client-side code must be updated to catch the new error type for proper handling of user cancellation. (f6d8d2e)

Networking & Security

  • ⚠️ BREAKING BREAKING CHANGE: The signature for StoragePartitionImpl::RevokeNetworkForNoncesInNetworkContext and its underlying Mojo interface has been changed. Callers must now pass a network::ConnectionAllowlists struct instead of a set of strings, breaking existing integrations. (996bdae)
  • ⚠️ BREAKING BREAKING CHANGE: Several expired InsecureDownloadSecurityStatus enum values have been removed from the public header components/download/public/common/download_stats.h. Any code referencing these enums will no longer compile. (7fa5463)
  • Security has been enhanced by adding validation for headers provided by renderer-initiated navigations. This helps prevent potential exploits from a compromised renderer process. (fdab935)
  • Performance of HTTP/2 has been improved by proactively creating a SpdySession on preconnect, and WebSocket throughput is enhanced by increasing the deflate stream chunk size from 4KB to 32KB. (f9e7758) (e0f9987)

Developer Tools

  • ⚠️ BREAKING BREAKING CHANGE: The DevTools DOM protocol has been updated. The isAdRelated boolean property on nodes is now replaced by a more detailed adProvenance object. Tools that inspect the DOM for ad-related information must be updated to parse this new structure. (f9c4f88)
  • The WebAuthn domain in the Chrome DevTools Protocol has been updated to support hmac-secret and hmac-secret-mc options for virtual authenticators, improving testing capabilities. (012c80d)

Autofill & Extensions

  • ⚠️ BREAKING BREAKING CHANGE: The AutofillManager::Observer interface has been modified. The signature for OnSuggestionsHidden now includes a SuggestionHidingReason, which will break existing observer implementations. (95237ac)
  • ⚠️ BREAKING BREAKING CHANGE: Several unused ServerFieldType enum values have been removed from components/autofill/core/browser/field_types.h and the autofill_private extensions API. Code referencing these types will no longer compile. (73deb77)
  • The chrome://policy page has been updated to display the status of extension install policies, providing more transparency for administrators. (49588c7)

Build & Test Infrastructure

  • On Windows, installer crash reports will now include logs from the updater, providing more context for diagnosing installation failures. (a8f6993)
  • A common MockSessionSyncService has been introduced to reduce code duplication and simplify the creation of sync-related unit tests. (ae7c922)
  • The base library has been updated to encourage the use of try_emplace for more efficient map insertions, preventing unnecessary object construction in many cases. (ca804d1)
