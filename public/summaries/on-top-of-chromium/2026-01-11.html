No major breaking changes were identified. Key updates include enabling a new printing API feature by default, introducing a feature flag for the upcoming 'Skills' feature, and various updates to the base library and third-party dependencies.

APIs & Extensions

  • The ApiPrintingMarginsAndScale feature for the Extensions Printing API is now enabled by default, allowing extensions to control margins and scaling during print jobs. This was previously behind a feature flag. (899baf6)
  • A race condition in the experimental WebNN API has been fixed. The change ensures that a context loss event is handled immediately, preventing reads from an invalid context. (6b088b9)

Browser & UI

  • A new feature flag, kSkills, has been introduced to control the "Skills in Chrome" feature, which is currently under development. (452f88b)
  • The sync settings page (chrome://settings/people) has been updated to show an informational message when a user has reached their bookmark limit. (93ac99d)

Core & Services

  • The //base library has been updated to include a new contains() member function for small_map, variant_map, and cstring_view, providing a more convenient way to check for element existence. This is an additive change. (34771ec)
  • An internal refactoring within the filesystem service (components/services/filesystem) updated its implementation to use modern span-based file APIs with no functional changes. (2be5a1c)

Dependencies & Infrastructure

  • Several third-party Rust crates have been updated to their latest patch or minor versions, including clap, syn, serde_json, libc, and indexmap. (3c9a2c8) (6eac6bc) (278e3d7) (0341300) (5110833) (28454d3) (c4100d6) (25720b8)
  • Routine maintenance includes updates to translation strings and histogram metadata. (d3d37d3) (a1ed230)
