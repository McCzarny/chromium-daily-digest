Chromium Digest: 2026-06-27

Overview

This update digests 63 commits, detailing security hardening, process isolation enhancements, network reliability fixes, and desktop UI modifications on Windows and macOS. Highlighted changes include a Rust migration for child process security mapping, QUIC client stream fixes, and robust visual layout bounds enforcement for WebApp prompts.

Security, Cryptography, and Process Isolation

  • Migrated the Origin-Agent-Cluster (OAC) states by BrowsingInstance mapping into Rust to enforce compile-time verification of process-isolation states. (4909829)
    SecurityBrowser
  • Extended the unexportable key service to support hardware-backed attestation key pools alongside generic signing keys. (b265430)
    CryptoSecurity
  • Added message envelope size limits for CBOR-formatted messages received over DevTools connections from potentially compromised renderers. (f8c6849)
    DevToolsSecurity
  • Updated Safe Browsing download protection proto declarations to enable capturing local antivirus program deployment statuses. (9dc8cff)
    SecurityDownloads

Network Stack & Media Operations

  • Fixed a use-after-free vulnerability by properly unregistering the Http3DatagramVisitor inside QuicChromiumClientStream::Handle destructor. (d6b9f8d)
    NetworkSecurity
  • Disabled Direct3D video processing (MFVP) on Windows Qualcomm platforms to work around hardware-accelerated scaling instabilities. (f3a8c14)
    MediaGPUWindows

Desktop UI, Sync, & WebApp Features

  • Replaced the static minimum bounds for PWA dialogs with custom shrinkage thresholds to auto-abort presenting dialogs if clipped on constrained windows, preventing identity-spoofing attacks. (60b5af5)
    UISecurityBrowser
  • Added a phone-based cross-device sign-in promotion option into the desktop profile menu on DICE-supported browser configurations. (19e47c6)
    ProfilesUISync
  • Introduced a new bubble controller API inside Autofill payments UI to re-engage users with card configuration offerings. (d2c8210)
    PaymentsUIAutofill
  • Enabled theme synchronization across linked mobile and desktop clients by registering the corresponding Sync controllers on Desktop platforms. (ea43628)
    SyncThemesBrowser
  • Fixed setting synchronization and state restoration issues within Reading Mode side panel sessions when switching between multiple tabs. (49531cc)
    UIBrowser

System Architecture & Code Health

  • Automated the migration of container-nested raw pointer fields (such as std::vector elements) to standard raw_ptr wrappers using clang-plugin tools. (261fed0)
    Base
  • Promoted the LoadAndVerifyModelOffThread utility to a public function, simplifying model integration and testing. (f0e51a1)
    AIBase
  • Moved Desktop Extensions browser tests out of the main chrome/test monolith and into isolated subdirectory build targets. (1d836b9)
    BuildExtensionsTesting
