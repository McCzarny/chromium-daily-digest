Week 52 of 2025 (2025-12-21 to 2025-12-27)

Chromium Weekly: 2025 Week 52

Overview

A significant week of API modernization and cleanup, this summary covers 529 relevant commits out of 852 total across 7 days. The week was dominated by a large number of breaking changes aimed at improving type safety and removing obsolete code, particularly within the Identity Manager, Task Manager, and Privacy Sandbox components. Other notable updates include functional improvements to the Extensions platform and continued development of the Vertical Tabs UI.

API Modernization & Breaking Changes

  • ⚠️ BREAKING BREAKING CHANGE: The task_manager::TaskManagerInterface API has been modernized to use base::ByteSize and std::optional. Methods like GetMemoryFootprintUsage and GetProcessTotalNetworkUsage now return std::optional<base::ByteSize> instead of using sentinel values, requiring embedders to update their code to handle the new types. (c89f2a8) (3010a9b)
  • ⚠️ BREAKING BREAKING CHANGE: A major refactoring of the Identity Manager API has made several members of AccountInfo private, requiring the use of new getter methods. Additionally, the components/signin/public/identity_manager/scope_set.h header and specific signin::AccessTokenFetcher constructors have been removed. (d2acdce) (454a9ad) (66263c5) (352eb26)
  • ⚠️ BREAKING BREAKING CHANGE: Several obsolete Privacy Sandbox classes have been removed, including TrackingProtectionSettings, its observer, and the TrackingProtectionOnboarding class. Embedders relying on these for managing tracking protection settings or onboarding flows will be broken. (36e492c) (98049c0) (18e568e)
  • ⚠️ BREAKING BREAKING CHANGE: Core browser and UI utilities have been updated. BrowserList::empty() is removed in favor of GlobalBrowserCollection, ui::FormatBytes now requires base::ByteSize, and base::TraceLog::IsEnabled() is replaced by base::TrackEvent::IsEnabled(). The C++23 std::to_underlying now replaces the base:: equivalent. (cef649e) (eae5d29) (5051d64) (271bd17)
  • ⚠️ BREAKING BREAKING CHANGE: The UserManager::Observer contract has changed to provide default empty implementations for its methods, which may alter runtime behavior for existing observers. Separately, the return type of PrerenderHostRegistry::CreateAndStartHost has been changed from FrameTreeNodeId to PrerenderHostId. (063bace) (8eacbc5)
  • ⚠️ BREAKING BREAKING CHANGE: The public methods for raw cookie access, such as GrantReadRawCookies, have been removed from ChildProcessSecurityPolicyImpl. In UI views, the virtual methods HandleWriteTextToClipboard and AllowStartDragEvent have been removed from views::TextfieldController. (41b3b0a) (fbef187)

Extensions

  • The sidePanel.open API now supports WINDOW_ID_CURRENT, allowing an extension to open a side panel in the currently active window without querying for its ID. Additionally, the runtime.onMessage event listener schema was updated to officially support returning a Promise for asynchronous responses. (beede36) (0d1975a)
  • ⚠️ BREAKING BREAKING CHANGE: The public method GetAssociatedWebContents() has been removed from extensions::ExtensionFunctionDispatcher, which may impact embedders that use it. (d74add7)
  • Work continues to align input.ime.onKeyEvent key values with the DOM specification (e.g., "Ctrl" to "Control"). The change is being developed behind the InputImeKeyboardEventDataToDOMSpec feature flag to allow for testing before it becomes default behavior. (e3e9b21) (1473840)

Browser UI & Features

  • The Vertical Tabs feature now supports drag-and-drop for reordering tabs. The underlying interface for updating loading animations has also been implemented. (96affd4) (1267e59)
  • The Print Preview dialog has been enhanced with an "actual size" option in the PDF scaling dropdown, allowing users to print documents without any scaling. (abdd4a8) (b13ef07)
  • For Progressive Web Apps on Windows, the Window Controls Overlay (WCO) toggle state is now managed on a per-window basis, fixing a bug where the state was incorrectly shared across all windows of the same app. (8e2fb48)

Security, Networking & Enterprise

  • To improve stability, the renderer process will now be terminated if it sends invalid arguments when creating an IndexedDB cursor. Renderer hang detection has also been made more accurate with a new PingMainThread IPC probe. (c903934) (e74ee98)
  • ⚠️ BREAKING BREAKING CHANGE: A dependency roll of the QUICHE library has removed several deprecated methods from the public quic::ParsedQuicVersion API in favor of a single, unified IsIetfQuic() method. (c5b2270)
  • ⚠️ BREAKING BREAKING CHANGE: The SafeBrowsingUIHandler has been componentized into an abstract base class and a concrete implementation, requiring embedders to update instantiation from SafeBrowsingUIHandler to SafeBrowsingContentUIHandler. (7341663)
  • Two new enterprise policies, GeminiActOnWebAllowedForURLs and GeminiActOnWebBlockedForURLs, have been added to provide administrators with URL-based control over the "Act on this page" AI feature. (0c67554)
