Chromium Digest: 2026-06-10

Overview

This digest summarizes 654 commits for 2026-06-10, with 516 commits identified as relevant to desktop (Windows/macOS) embedders. The period spans from commit (68d9c3f) to (353aa2d), focusing on browser architecture, security hardening, and API stability.

Breaking Changes & API Evolution

  • ⚠️Removed GetPrintServersConfig, ChoosePrintServers, and AddPrintServerObserver from the LocalPrinter interface. (7ba8f73)
  • ⚠️Removed the virtual OnInputIgnored method from RenderWidgetHostDelegate and RenderInputRouterDelegate. Embedders overriding this must update their implementations. (ccaf056)
  • ⚠️Changed the default value of the position-anchor CSS property to normal, impacting layout resolution for custom stylesheet assumptions. (3357456)
  • Converted webapps::LaunchParams from a struct to a class, requiring migration to new getter/setter methods. (7b413c3)
  • Renamed EntityTypeName to EntityType within the Autofill personal context API. (4854dfd)

Security, Identity & Networking

  • Hardened DIAL protocol parsing by rejecting dot-segments to prevent directory-confinement escapes. (d3ec8b1)
  • Introduced KeyRepository in UnexportableKeyService to centralize device-bound session key management. (22fe77f)
  • Offloaded NetworkContext creation to the ThreadPool to reduce main-thread startup latency. (3800247)
  • Skia migrated to raw_ptr via SK_USE_PARTITION_ALLOC to mitigate UAF exploits. (69e1d69)

UI, Extensions & Infrastructure

  • Modularized chrome_layout_provider.h and chrome_typography_provider.h into dedicated //chrome/browser/ui/views:layout_provider targets to improve compilation times. (4f3c2c3)
  • Decoupled NetModule into //net/base/module requiring updates to existing include paths. (de9fc16)
  • Implemented auto-hide logic for the Bookmark Bar on the New Tab Page with integrated IPH support. (c40d91d) (3d9b2ae)
