Week 53 of 2025 (2025-12-28 to 2025-12-31)

Chromium Weekly: 2025 Week 53

Overview

This week saw a flurry of significant, and often breaking, API refactorings across the codebase. Approximately 75 relevant commits out of 298 total across 4 days introduced major changes to tab management, extension APIs, and security components, requiring careful review by embedders. Key themes include the modernization of the TabStripModel, removal of deprecated privacy and Safe Browsing methods, and behavioral changes to the Declarative Net Request API.

APIs & Core Internals

  • ⚠️ BREAKING BREAKING CHANGE The TabStripModel and its observer have been significantly refactored. TabStripModelObserver methods were renamed (e.g., TabChangedAt is now OnTabChangedAt), and TabStripModel::selection_model() now returns a new TabStripModelSelectionState object instead of a ListSelectionModel-compatible adapter, breaking all code that interacted with the previous index-based selection model. (fdb26aa) (7791541) (bd1c829)
  • ⚠️ BREAKING BREAKING CHANGE The public Mojo binding headers have been modernized to use C++20 concepts instead of SFINAE for trait detection. This is a breaking change for advanced embedders who specialize or interact with the previous trait implementations. (694c89f)
  • ⚠️ BREAKING BREAKING CHANGE The searchbox.mojom interface was refactored, renaming several types by removing the Stub suffix (e.g., SearchContextStub is now SearchContext). This breaks clients using the old type names. (1169c88)
  • ⚠️ BREAKING BREAKING CHANGE The signature for InputEventObserver::OnInputEvent in content/public/browser/ has changed to require a new InputEventSource parameter. All implementations of this public observer must be updated. (a9bd994)
  • The TranslationAPI feature flag has been removed, enabling the window.chrome.translation API by default and making it consistently available. (fdaa2fc)

Extensions

  • ⚠️ BREAKING BREAKING CHANGE The ExtensionFunctionDispatcher::Delegate public interface was significantly altered. The virtual methods GetAssociatedWebContents() and GetVisibleWebContents() have been removed, breaking embedder implementations. (9885010) (dc15d34)
  • ⚠️ BREAKING BREAKING CHANGE Several public methods have been removed from the ExtensionTabUtil utility class, including GetWindowIdOfGroup(), GetBrowserWindowTypeText(), and GetActiveTab(). Code relying on these methods will no longer compile. (f13ff20) (e6fa848)
  • A significant behavioral change was made to the Declarative Net Request (DNR) API: rules will no longer be applied to requests originating from <webview> tags. This may impact extensions that were relying on this functionality. (2862890)
  • A use-after-free crash in the extension's GinPort during service worker termination has been fixed by invalidating ports when the context is destroyed. (b84da97)

Security & Privacy

  • ⚠️ BREAKING BREAKING CHANGE Deprecated Safe Browsing hit reporting methods, MaybeReportSafeBrowsingHit and CreateAndSendHitReport, have been removed from the BaseUIManager and SafeBrowsingUIManager classes. (cbeca2e)
  • ⚠️ BREAKING BREAKING CHANGE The public header components/privacy_sandbox/tracking_protection_prefs.h and its registration function have been deleted. Remaining preferences were consolidated into privacy_sandbox_prefs. (dd66256)
  • ⚠️ BREAKING BREAKING CHANGE The public method can_use_chrome_ip_protection() has been removed from the AccountCapabilities class in the Identity Manager component. (47c3d62)
  • Development on Device Bound Session Credentials (DBSC) continues with the addition of a new chrome://unexportable-keys-internals debug page and new mojo representations for debugging. (814bcf8) (57ba3ce) (f7316aa)

Browser & UI

  • ⚠️ BREAKING BREAKING CHANGE The BinaryUploadService header, used for enterprise content scanning, has been moved from chrome/browser/safe_browsing/ to components/enterprise/connectors/core/, breaking existing build includes. (eb3d823)
  • ⚠️ BREAKING BREAKING CHANGE The ContextualTasksContextController class, along with its factory and test support files, has been completely deleted from the codebase. (8858a40)
  • ⚠️ BREAKING BREAKING CHANGE The public method ReadAnythingController::ShowUI has been renamed to ReadAnythingController::ShowSidePanelUI to more accurately reflect its function. (599fcf8)
  • New mojo methods have been added to the updater_service.mojom interface to allow querying of the updater's state and policies. (18ef7c7)
