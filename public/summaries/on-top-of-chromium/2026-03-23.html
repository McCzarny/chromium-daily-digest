Summary for 2026-03-23 on branch 'main'

Chromium Digest: 2026-03-23

Overview

This summary covers 578 commits on the main branch for March 23, 2026, with 438 identified as relevant to developers building on Chromium. The day was marked by an unusually high number of breaking changes across core APIs, networking, and UI components, requiring significant updates for embedders. Key non-breaking work included progress on post-quantum cryptography and UI enhancements for Vertical Tabs. The first relevant commit was (12284ce) and the last was (db815f5).

Breaking Changes

  • ⚠️ BREAKING The public API in ui/base/clipboard/clipboard.h has been modified. Methods like WriteBookmark() and ReadBookmark() have been renamed to WriteURL() and ReadUrl() respectively. Embedders using the clipboard for URL/bookmark data must update their method calls. (624e217)
  • ⚠️ BREAKING Multiple core API signatures have been changed, requiring code updates. This includes: the removal of the 2-parameter overload for content::WebContents::IgnoreInputEvents; renaming OnComplete() to OnBannerShown() in webapps::AppBannerManager::Observer; adding a force_to_proceed parameter to the BeforeUnload Mojo API; and changing Browser* parameters to BrowserWindowInterface* in chrome/browser/ui/browser_commands.h. (05268bd) (e89a8d0) (2cb8be8) (14ee99d)
  • ⚠️ BREAKING The AccountSettingService has been moved from components/autofill to a new components/account_settings component. This changes its namespace from autofill to account_settings, breaking includes and direct references. (a1b05c5) (5c162c5)
  • ⚠️ BREAKING The core networking Mojo interface network.mojom.NetworkContext.CreateWebSocket has a new network_restrictions_id parameter. Direct callers must be updated to provide this new value. (0f1662a)
  • ⚠️ BREAKING Navigation queueing is now permanently enabled, and the kQueueNavigationsWhileWaitingForCommit feature flag has been removed. This represents a fundamental behavioral change in how concurrent navigations are handled. The UndoCommitNavigation() IPC method was also removed from content/common/frame.mojom. (369b2d9)
  • ⚠️ BREAKING Several public APIs and components have been refactored or removed. This includes: renaming OnTabsClosed to OnNodesClosed in TabStripService; removing the PageActionIconType::kCollaborationMessaging enum; removing GetOnDeviceCapabilities from the OnDeviceCapability class; and changing the constructor for components/sync_device_info/DeviceInfo. (73c9765) (2c4a110) (6e00555) (505ebd5)
  • ⚠️ BREAKING File paths for key resources have been changed, which will break builds with hardcoded include paths. x509_certificate_model.h moved from chrome/common/net to components/certificate_model, and cws_item_service.proto moved from //chrome to //extensions. (35ed0b5) (74dc059)
  • ⚠️ BREAKING The command-line switch --enable-webgl-image-chromium has been renamed to --enable-webgl-shared-image. Scripts and configurations using the old switch must be updated. (1d5f025)

APIs & Core

  • A new LocationIconInterface has been introduced to decouple location bar chip controllers from the native Views implementation. This abstraction will enable future WebUI-based location bar components. (8222d0b)
  • As part of Project Bedrock, the ProfileManager's dependency on BrowserListObserver has been removed to reduce coupling with the Browser object. (7427fd2)
  • A new SidePanelUIProvider class provides a cross-platform way to retrieve the SidePanelUI interface, simplifying code that interacts with the side panel. (dc6cacd)
  • Navigation-related protocol buffers have been moved from session_specifics.proto to a new tab_navigation.proto. This refactoring enables reuse for features like Send-Tab-To-Self history without depending on the entire Sessions data type. (84f565f)

Networking & Security

  • Experimental support for the post-quantum signature algorithm ML-DSA (variants 44, 65, and 87) has been added to WebCrypto. The feature is gated by the WebCryptoPQC runtime flag. (2da14fd)
  • A new enterprise policy, MaxConnectionsPerProxyForWebSocket, has been added. This allows administrators to control the WebSocket proxy connection limit, providing more granular control over network resources. (1c530cb)
  • The Brotli compression library was updated to fix a decoding hang on certain inputs. (224140d)
  • A bug in the file download access prevention mechanism that could cause downloads of highly compressible files to fail has been fixed. (81d22a5)

Browser UI (Windows/macOS)

  • The Vertical Tabs feature continues to be refined, with improvements to focus management after moving tabs, a fix for the drop indicator arrow's position, and the ability for the tab strip region to expand on hover. (5d9dd66) (15a6396) (885030e)
  • A crash on macOS in TabStrip::IsRectInWindowCaption that could occur when switching to vertical tabs has been fixed by adding a null check. (703c279)
  • Crashes in the Profile Picker and other signed-out flows were fixed by ensuring WebContentsModalDialogManager is correctly initialized for the associated WebContents. (961e217) (6d519a6)
  • Payment Handler windows will now respect the "Auto-open DevTools for pop-ups" setting, improving the developer experience for debugging payment flows. (6b7477f)

Build & Test Infrastructure

  • The autotest script was improved with a new --run-related flag to heuristically find and run tests related to locally modified files, simplifying the developer workflow. (7d04f66) (0ee936d) (df2610b)
  • The linux-afl-asan-rel builder, which has been broken for over a year, was removed from the build infrastructure, along with its associated afl GN config. (45a34c1)
  • The Siso build tool was updated to version 1.5.7. (32939b2)
