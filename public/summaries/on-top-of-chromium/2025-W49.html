Week 49 of 2025 (2025-11-30 to 2025-12-06)

Chromium Weekly: 2025 Week 49

Overview

This week was defined by a monumental number of breaking changes across the Chromium codebase, requiring significant attention from downstream projects. A total of 872 relevant commits were made out of 1201. The most impactful change was the upgrade of the default toolchain to C++23. Additionally, major API refactorings occurred in core identity, media, security, and UI components, alongside the complete removal of the experimental IP Protection and Identifiability Study features.

Build System & Toolchain

  • ⚠️ BREAKING BREAKING CHANGE: The default C++ standard for the Chromium build has been upgraded to C++23. Downstream projects must update their build configurations and code to be C++23 compatible. (5047a65)

Core APIs & Breaking Changes

  • ⚠️ BREAKING BREAKING CHANGE: The AccountInfo::hosted_domain member has been made private across multiple commits. Embedders must now use the GetHostedDomain() accessor method, which returns a std::optional. (0cc5bbc) (bb0a00d) (265e995) (cca5b70)
  • ⚠️ BREAKING BREAKING CHANGE: The media::VideoFrame API was heavily refactored. STORAGE_GPU_MEMORY_BUFFER was renamed to STORAGE_MAPPABLE_SHARED_IMAGE, and methods for accessing GPU memory buffers were renamed or removed in favor of a SharedImage abstraction. (9529e2b) (42fcda8) (db2f79e) (fdbf02d)
  • ⚠️ BREAKING BREAKING CHANGE: The public content::ContentBrowserClient embedder interface was changed. Virtual methods GetIpProtectionProxyBypassPolicy, GetUserAgentBasedOnPolicy, and IsUnpartitionedStorageAccessAllowedByUserPreference were removed. (a402ef2) (18b17a9) (fcea483)
  • ⚠️ BREAKING BREAKING CHANGE: Support for the --headless=old command-line switch has been completely removed. Automation scripts must be updated to use the new headless mode. (b18a8a9)
  • ⚠️ BREAKING BREAKING CHANGE: Several tab-related public APIs were removed or changed, including TabStripModel::GetTabCount() (replaced by count()), TabStripModelDelegate::ShouldDisplayFavicon, and the AllTabContentsesList class. (cce3eac) (cc15891) (4fd923f)
  • ⚠️ BREAKING BREAKING CHANGE: To improve build times, full headers were replaced with forward declarations in widely-used public headers like content_browser_client.h and navigation_handle.h, which may cause build failures in projects relying on transitive includes. (f1d6b69) (008a5c4)
  • ⚠️ BREAKING BREAKING CHANGE: The SimpleURLLoader callback signature was modernized to use std::optional<std::string>, and the deprecated BodyAsStringCallbackDeprecated was removed. (5711954) (a89abc5)
  • ⚠️ BREAKING BREAKING CHANGE: The public url::StringViewReplacements class has been removed and its functionality merged into url::Replacements. (d170193)

Security & Privacy

  • ⚠️ BREAKING BREAKING CHANGE: The experimental IP Protection (IPP) feature and its entire //components/ip_protection component have been removed from the codebase. (a402ef2) (41bfe9a) (c6fcfae)
  • ⚠️ BREAKING BREAKING CHANGE: The public API for WebAuthn passkeys was refactored. Several methods like GetUnShadowedPasskeys() and GetAllPasskeys() were removed from PasskeyModel in favor of a new unified GetPasskeys() method. (c1498b4) (a3f9ac7) (583d7e5) (2c8021b)
  • ⚠️ BREAKING BREAKING CHANGE: Public headers for the WebAuthn (FIDO) API have been moved from //device/fido/ to a new //device/fido/public/ directory, breaking existing include paths. (e3c9acb) (0615b88) (8735bd5)
  • ⚠️ BREAKING BREAKING CHANGE: The UserAgentReduction enterprise policy and its associated helper functions in embedder_support have been removed. (64f74a0)
  • The experimental Identifiability Study (Privacy Budget) feature has been removed from the codebase, cleaning up its associated mojo interfaces and implementation. (db00ab2) (0d71dc3) (b770092)

UI & Views Framework

  • ⚠️ BREAKING BREAKING CHANGE: The primary tab strip UI class TabStripViewInterface has been renamed to TabStripRegionView, requiring updates for any code that interacts with the tab strip view. (16f1c19)
  • ⚠️ BREAKING BREAKING CHANGE: The viz::TransferableResource graphics struct was refactored. It no longer contains individual fields, which are now accessed through a gpu::ClientSharedImage. This breaks code that directly constructs or accesses its previous members. (bbb801f)
  • ⚠️ BREAKING BREAKING CHANGE: In ui/views, view builder factory logic has been moved from the core ui/views/view.h header into ui/views/metadata/view_factory.h, breaking includes that relied on the previous location. (a878963)

Networking

  • ⚠️ BREAKING BREAKING CHANGE: The signature of net::disk_cache::CreateCacheBackend has changed to include a new cache_encryption_delegate parameter, affecting embedders who create their own disk cache backends. (f7ba4f3)
  • The network service's client certificate cache will now be flushed when the AutoSelectCertificateForUrls policy changes, allowing new rules to take effect without a browser restart. (5a79971)
  • A bug was fixed where HTTP 425 (Too Early) responses were incorrectly retried for all idempotent requests. Retries will now only occur if TLS early data was actually sent. (1a07b4e)

Extensions

  • ⚠️ BREAKING BREAKING CHANGE: Several enums in the extensions API, such as CreatePerContextData and ScriptInjection::InjectionResult, were converted to enum class. Consumers must update their code to use scoped names. (a02b50d)
  • Extension service worker registration will now be retried in more transient error cases, such as kErrorDiskCache, improving the reliability of extension startup. (152316e)
