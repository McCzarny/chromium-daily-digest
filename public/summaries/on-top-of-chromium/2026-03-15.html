Chromium Digest: 2026-03-15

Overview

This digest covers 8 commits, with a focus on changes relevant to embedders on Windows and macOS. One major breaking change was identified in the base macOS utilities, where a public enum for system settings was modified. Other updates include UI improvements for Vertical Tabs and internal testing enhancements.

Browser APIs (macOS)

  • ⚠️ BREAKING BREAKING: The public base::mac::SystemSettingsPane enum, used with OpenSystemSettingsPane to open macOS System Settings, has been updated. Several enum values were removed and replaced to align with upcoming macOS versions. For example, kDateTime is now kGeneral_DateTime, and kPrivacySecurity_Pasteboard is now kPrivacySecurity_PasteFromOtherApps. Embedders on macOS using the old enum values will encounter build failures. (6c38bdc)

UI & UX

  • The Vertical Tabs feature received animation improvements to prevent visual jank. The changes ensure that animating tabs are correctly scrolled into view, especially in overflow scenarios, and that the new tab button remains flush with the scroll view during animations. (f303bc0) (471bde2)

Testing

  • The TabSearchPageHandler unit tests were refactored to reset the handler during teardown, preventing spurious notifications that could mask test expectations. Separately, a dangling pointer was fixed in OptOutBlocklistTest to resolve a memory safety issue. (b75ca23) (60e4de8)
