Chromium Digest: 2026-01-23

Overview

This summary covers 531 relevant commits out of 672 total from the main branch on January 23, 2026. The day was marked by a significant number of breaking API changes, including a major mechanical rename of base::Value::Dict and base::Value::List, numerous interface modifications in the UI, extensions, and security layers, and several file path and header relocations. The changes span from (dd4a3fe)" target="_blank" rel="noopener noreferrer" class="text-sky-500 hover:text-sky-300 font-mono">((dd4a3fe)) to (4b3959a)" target="_blank" rel="noopener noreferrer" class="text-sky-500 hover:text-sky-300 font-mono">((4b3959a)).

Breaking Changes

  • ⚠️ BREAKING The widely used base::Value::Dict and base::Value::List have been renamed to base::DictValue and base::ListValue respectively. This is a mechanical but breaking change that requires embedders to update all usages of these fundamental data types from the base library. (8e25bad) (42b022b) (d673b3a)
  • ⚠️ BREAKING A new pure virtual method, GetGroupIdForSavedGroup, has been added to the public sessions::LiveTabContext interface. Embedders who implement this interface for session restoration must add an implementation for this new method. (1ae9a24)
  • ⚠️ BREAKING The extensions::ManagementPolicy::Provider::UserMayInstall method signature has been changed from a synchronous bool return to an asynchronous function that uses a callback. Custom providers must be updated to the new async pattern. (b79a8c6)
  • ⚠️ BREAKING In content::ContentBrowserClient, the method ShouldOverridePrivateNetworkRequestPolicy has been renamed to ShouldOverrideLocalNetworkAccessRequestPolicy. Embedders implementing this method must update to the new signature. (2c60d0e)
  • ⚠️ BREAKING Several public virtual methods have been removed from the TabStripController and TabSlotController interfaces, including CanShowModalUI, ShowModalUI, and IsTabPinned. Custom UI implementations subclassing these controllers will need to be updated. (e2eb5ce)
  • ⚠️ BREAKING The core browser navigation functions chrome::CanGoBack and chrome::GoBack have been modified to incorporate "back-to-opener" logic, altering the fundamental behavior of the back button. (81bc0fc)
  • ⚠️ BREAKING The constructor for ReloadButton (chrome/browser/ui/views/toolbar/reload_button.h) now requires a Browser* instance, breaking custom toolbar code that directly instantiates this class. (f10c65f)
  • ⚠️ BREAKING The public headers split_tab_id.h and split_tab_visual_data.h have been moved from components/tabs/public/ to a new component components/split_tabs/, breaking include paths. (fd58fdc)
  • ⚠️ BREAKING The content setting ContentSettingsType::DURABLE_STORAGE has been renamed to ContentSettingsType::PERSISTENT_STORAGE. Code managing this permission must be updated. (9ca1f26)
  • ⚠️ BREAKING Several file access methods in the public content::ChildProcessSecurityPolicy interface (e.g., GrantReadFile) have been updated to use the strongly-typed ChildProcessId instead of an int. (4396132)
  • ⚠️ BREAKING The method GetPasskeyByUserId has been removed from the public PasskeyModel interface. Consumers should now use GetPasskeys() and filter the results manually. (1f9a2d6)
  • ⚠️ BREAKING The virtual method GetCurrentFormInteractionsFlowId() has been removed from the AutofillClient interface. Embedders must remove this method from their implementations. (4d268be)
  • ⚠️ BREAKING The public class GpuMemoryBufferVideoFramePool in //media has been renamed to MappableSharedImageVideoFramePool. Downstream code must be updated to use the new class name. (058f840)
  • ⚠️ BREAKING The SetIsCrossOriginIsolated method has been removed from the public content::mojom::Renderer mojom interface, breaking code that uses it for browser-renderer communication. (6686a4a)
  • ⚠️ BREAKING The borderless_url_patterns field has been removed from the WebApp protobuf definition and its corresponding C++ structs, impacting code that interacts with Web App installation or persistence. (51b7656)
  • ⚠️ BREAKING The public GPU IPC headers gpu/ipc/common/gpu_command_buffer_traits.h and gpu/ipc/common/gpu_command_buffer_traits_multi.h have been removed as part of a migration to Mojom. (d34682a)
  • ⚠️ BREAKING The public method AnyActionHasCurrentSiteAccess() has been removed from the ExtensionsToolbarViewModel public API. (ea548b0)

Browser & UI

  • The Tab Search button in the toolbar is now enabled by default, altering the default appearance of the browser's top chrome. (eeb683f)
  • UI surfaces related to Autofill and Payments have been updated to use Google Wallet branding and icons, replacing the previous Google Pay branding. (058fc79) (7c4a770)
  • The session restore mechanism now preserves the collapsed state of tab groups when a window is restored. This change also introduced a breaking API change to sessions::LiveTabContext. (1ae9a24)
  • On macOS, starting application audio capture on an app without an audio source will now generate a silent stream instead of an error, aligning its behavior with Windows. (625a098)
  • The PWA window.setResizable API now returns a promise that resolves after the window's resizability state has been successfully changed, providing more reliable behavior. (40fe819)
  • On macOS, Chrome will now exit if the first-run dialog is dismissed without acceptance. (f4d0239)
  • The google-chrome:// URI scheme is now registered on macOS for stable builds, aligning it with Windows behavior. (d735f29)

Extensions

  • A dangling pointer bug in GinPort has been fixed, improving the stability of extension messaging by ensuring pointers are cleared when a port is disconnected. (200b225)
  • The identity.launchWebAuthFlow API now supports showing an infobar in the authentication pop-up, which can provide users with more context during the flow. (0e63868)
  • A bug has been fixed to better handle errors when a user attempts to load an unpacked extension with a non-existent manifest file. (f58b141)

Security, Policy & Networking

  • The asynchronous methods for the Supervised User URL filter have been migrated to the SupervisedUserUrlFilteringService, resulting in a breaking API change for direct callers. (bd2636d)
  • On macOS, Data Controls policies are now enforced for drag-and-drop operations, preventing data exfiltration from managed contexts. (fffd35b)
  • The kill-switch feature flags kPdfEnforcements and kSandboxedFrameEnforcements have been removed, making the associated security policies for PDFs and sandboxed frames permanent. (3e3051e)
  • The LensCameraAssistedSearchEnabled policy is being deprecated in favor of the new SearchContentSharing policy. (58200f7)

Developer Tools & Testing

  • The DevTools Protocol command Target.createTarget now accepts an optional focus parameter. This allows automation tools to create a new foreground tab without bringing the entire browser window to the front. (89de898)
  • New build-time CSS linting checks have been enabled for WebUI, enforcing a consistent style (e.g., single quotes) across WebUI components. (cac5b1b)
  • Recent changes to the activity_reporter component, which communicates browser usage to the updater on Windows and macOS, were reverted, temporarily restoring previous reporting logic. (0322fa2) (dd4a3fe)
