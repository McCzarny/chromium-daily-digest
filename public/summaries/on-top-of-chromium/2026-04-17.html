Breaking Changes

  • ⚠️ BREAKING The entire synchronous OSCrypt backend has been removed. The components/os_crypt/sync directory, including the OSCryptMocker for testing, is deleted. Embedders must migrate to the asynchronous os_crypt_async APIs. On Windows, calls to OSCrypt::Init must be replaced with os_crypt_async::Init. (a4cab4a) (21b80f4) (51493b3)
  • ⚠️ BREAKING The public method ToolbarButtonProvider::GetAvatarToolbarButton() has been removed. Embedders must migrate to the new GetAvatarToolbarButtonInterface() method to access the avatar button view. (6c9ccc1)
  • ⚠️ BREAKING The deprecated public method net::HashValue::DeprecatedFromString has been removed. Code must be updated to use the net::HashValue::FromString static factory method, which returns an std::optional. (104e45e)
  • ⚠️ BREAKING The SharedImageInfo and SharedImageMetadata structs have been moved from gpu/command_buffer/client/ to gpu/command_buffer/common/shared_image_info.h. Embedders using these GPU client interfaces must update their include paths. (59eee09)
  • ⚠️ BREAKING The virtual method CreateChromeMetadataPacketRecorder has been reverted and removed from the public content::TracingDelegate interface. Embedders implementing this delegate will fail to compile. (956f380)
  • ⚠️ BREAKING The signature for PrePrefetchService::Create in content/public/browser/pre_prefetch_service.h has been changed. Two bool parameters are now std::optional<bool>, which is a source- and binary-incompatible change. (7a24f21)
  • ⚠️ BREAKING The method signature for OnMediaMetadataChanged in content/browser/media/media_web_contents_observer.h has changed, swapping the order of has_audio and has_video parameters. Downstream overrides will break. (1a7e135)
  • ⚠️ BREAKING The passage_embeddings::Embedding component API no longer has a public Dimensions() method. Clients must update their code to use GetData().size() instead. (49a6256)
  • ⚠️ BREAKING The blink::mojom::DevicePostureProvider mojo interface has been changed. OverrideDevicePostureForEmulation and DisableDevicePostureOverrideForEmulation were removed, breaking consumers in testing and emulation scenarios. (829bf4e)
  • ⚠️ BREAKING Extensions using Declarative Net Request (DNR) to redirect to or from file:// URLs will now fail unless the extension has been granted the "Allow access to file URLs" permission. This may break existing extension functionality. (53caa53)

Security

  • The extensions EventRouter now performs stricter validation on listener registration. Renderer processes that attempt to register listeners with invalid parameters (e.g., mismatched extension ID) will now be terminated. (5cca17e)
  • To mitigate a potential vulnerability, a timeout has been added to the Variations Service seed fetch request, preventing an on-path attacker from delaying the response to manipulate the browser's network time. (ad917ff)
  • For Device Bound Session Credentials (DBSC), the UnexportableKeyService API was updated for improved type safety. Callbacks now return a role-specific UnexportableSigningKeyId to prevent key misuse at compile time. (402bd74) (4efe919)
  • On macOS, depth limits have been added when processing pasted bookmark data to prevent a stack overflow crash from maliciously crafted pasteboard content. (c5f67dc)
  • The CorsURLLoader now blocks modification of the Origin header during a cross-origin redirect, hardening against potential spoofing attacks. (fd71efb)
  • On Windows, Module Tampering Protection is now enabled for Code Integrity Guard (CIG) processes to defend against IAT hooking by third-party kernel drivers. (60632ed)

Extensions

  • The precedence for which extension handles a given MIME type has been redefined with a new 3-tier rule. This may alter which handler is invoked in environments with multiple extensions handling the same MIME type. (acf2d3c)
  • The installation process for extensions driven by sync or policy has been updated to use the new update_client stack, where it previously defaulted to the old stack for the first install. (e585ba1)
  • Foundational work has been added to the Tabstrip API to support a future context menu. (c46e867)

Browser & Core APIs

  • A new virtual method, OnFederatedLogin, has been added to the WebContentsObserver interface, allowing observers to be notified when a federated login (FedCM) request completes. A default implementation is provided. (17d438b)
  • A new API, performance.speculations.preloads, has been added to expose preload measurement data on the Performance object. (2f2079d)
  • In the Chrome DevTools Protocol (CDP), the Target.TargetInfo object now exposes a parentId, providing the ID of the parent target (e.g., the frame that created a worker). (96647e3)
  • A new StartupCommandLineCollector class was introduced for enterprise reporting, allowing the collection of command-line switches at browser startup on desktop platforms. (380c663)

Browser UI

  • The Tab Search button in the Horizontal Tab Strip has been reverted to use a Chevron icon. The Magnifying Glass icon is now reserved for the Vertical Tab Strip. (14d4393)
  • In the Reading Mode side panel, nav elements are now hidden and default text decorations for certain semantic HTML tags have been removed to improve the reading experience. (a61f150) (750a791)
  • A fix was implemented to prevent significant overpainting and redundant paint scheduling on the AI page action icon when dragging a browser window. (835b00a)

Networking

  • Support for older, experimental crypto versions for Private State Token (PST) redemption has been removed. Only the stable PrivateStateTokenV1Voprf version is now allowed. (b57cd02)

Build & Testing

  • Modularization efforts continue, with components like notifications, subresource_filter, policy, and local_discovery being moved into their own build targets to improve build times and dependency management. (4ae8b1e) (e514a41) (70f09b8) (1b7e251)
  • The SyncTest::SignIn() helper method was updated to no longer enable history sync by default, better reflecting the default user experience in tests. (b91e14d)
