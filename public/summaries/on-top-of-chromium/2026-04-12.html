Extensions

  • ⚠️ BREAKING The virtual method GetPrefServiceForContext has been removed from the public ExtensionsBrowserClient interface. Embedders who provide their own implementation of this class must now use user_prefs::UserPrefs::Get(context) to retrieve the PrefService for a given BrowserContext. This change will break any downstream implementations that were overriding this method. (2a97ba7)

Browser UI

  • The browser UI received several updates, including performance optimizations for vertical tab animations and the addition of zoom bubble support for WebUI-based browsers. A bug was also fixed in the WebUI pinned toolbar to correctly handle non-pinnable actions. (43868b0) (efcc15d) (8768309) (9824296)

Payments

  • Additional logging was introduced for the Facilitated Payments feature to record whether a detected iframe shares the same origin as the main frame. This is an internal metrics change with no external impact. (e3ac2ce)
