Chromium Digest: 2025-11-22

Overview

This digest covers 46 commits from (f12153c) to (ccad76c). Key updates include two major breaking changes to the `views::DialogDelegate` and `signin_metrics::AccessPoint` public APIs. Other notable changes involve the introduction of an experimental DevTools command for AI content annotation and initial support for fullscreen in WebUI-based browsers.

UI Framework (Views)

  • ⚠️ BREAKING The views::DialogDelegate public API has been changed. The virtual methods SetTitleMargins() and SetFootnoteMargins() have been removed and replaced by a new SetFrameMargins() method that takes a FrameMargins object. This requires embedders who subclass DialogDelegate or call the removed methods to update their code. (3645cfd)

Identity & Authentication

  • ⚠️ BREAKING A recent change that added kSettingsYourSavedInfo to the public signin_metrics::AccessPoint enum has been reverted due to breakages. The enum value has been removed, which will break any client code that had already adopted it. (5d585f0)
  • The PasskeyUnlockManager has been updated to consider the readiness state of the secure enclave when determining if passkeys are locked. This internal logic change improves the accuracy of the passkey lock status. (14cc7eb)

Developer Tools

  • A new experimental DevTools command, Page.getAnnotatedPageContent, has been introduced to allow fetching of AI-annotated page content. This adds a new virtual method with a default implementation to the WebContentsDelegate public interface. (a2401fd)

Networking & Security

  • A new component updater, PKIMetadataFastpush, has been added to load Merkle Tree Certificate (MTC) data. This data will be used for Trust Anchor ID negotiation, controlled by the new kVerifyMTCs feature flag. (0fba4e9)

Browser Features

  • Support for fullscreen exclusive access has been implemented for WebUI-based browser windows (Webium), enabling use cases like video fullscreen. This involves a new WebUIBrowserExclusiveAccessContext and additions to the browser.mojom interface for this experimental feature. (502564c)

Build & Refactoring

  • A series of include cleanups were performed across the codebase, removing unneeded headers like base/functional/callback_helpers.h and forward-declaring classes like gfx::Range to reduce header dependencies and improve build times. These changes are internal and should not affect embedders. (adc897f) (cc7dd4e) (a3c6931) (5fb1c2e)
