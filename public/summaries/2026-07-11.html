Chromium Digest: 2026-07-11

Overview

This digest summarizes 46 commits analyzed from branch main. Key highlights include critical security hardening of Encrypted Client Hello (ECH) connections, zero-copy WebGPU graphics enhancements on Windows, and massive O(n) rendering optimization for bookmark migrations. The commits span between first commit (4eff2e9) and last commit (f0eda5f). No major breaking changes to public stable APIs were identified.

Network, Security & Platform Sandbox

  • Implemented strict-mode Encrypted Client Hello (ECH) validation within TLS and QUIC, returning ERR_STRICT_ECH_REQUIRED to prevent connection attempts when required configuration keys are missing, effectively shutting down potential cleartext fallback routes. (b3d083a)
  • Extended the low-level Windows process launcher LaunchProcessWithToken to accept AppContainer security capabilities, paving the way for sandbox isolation improvements in off-loaded network processes. (4eff2e9)
  • Aligned WebUSB wire-protocol mapping for RESERVED requests on Windows with USB specifications, and updated DeviceImpl IPC validation boundaries to explicitly block unexposed request types by default. (4a5d45b)
  • Eliminated raw C-style array parsing logic in EAC3 and AC3 audio demuxers by refactoring to bounds-checked std::array configurations, preventing potential buffer overflow risks. (4b209df)

Graphics & Web Platform APIs (Blink)

  • Updated Blink WebGL validation to implicitly terminate active Pixel Local Storage (PLS) sessions during calls that exit a render pass (e.g., framebuffer binding, pixel reading, and texture copying) in accordance with the latest ANGLE spec modifications. (f0baab0)
  • Prevented duplicate Web Serial connect and disconnect events across isolated V8 worlds by routing events via DOMWrapperWorld-bound SerialConnectionEvent containers. (86ba51e)
  • Optimized popover and menu accessibility keyboard configurations, correcting left/right arrow behaviors so they do not close active menulists when adjacent focus targets are absent. (a8ed3a3)
  • Introduced a minimum delay threshold of 10ms for speculative rules under moderate heuristic controls to mitigate potential timing leaks. (837b286)

Internal Optimization & Performance

  • Reduced bulk bookmark mutation overhead from O(n²) to linear O(n) by refactoring BookmarkEventTranslator to perform lazy, on-demand pre-mutation snapshot captures inside reordering and deletion hooks. (48225a2)
  • Added the READ_CONTROL flag inside Windows DuplicateHandle calls to allow WebGPU (Dawn) to directly map shared memory file mapping handles as native D3D12 heaps, enabling zero-copy buffer transfers. (4c84a83)

Tooling, Refactoring & WebUI Infrastructure

  • Upgraded internal WebUI TypeScript tooling support to v7.0.1-rc, fixing structural JSDoc typed warnings and providing TemplatizedDomNodes types configuration to resolve compilation gaps inside templated elements. (b5309ea) (3ce8a1d)
  • Continued migration of internal settings pages, searchable containers, and helper bubble components from Polymer to Lit. (610ed97) (feea616)
  • Simplified Mojo binding structures across WebUI components by purging multiple redundant, empty Page interfaces and transitioning to automated MojoHandlerProxyFactory implementations. (b33a890)
  • Deconstructed the monolithic CBOR parser library in Rust into clean modular components, introducing extended test suites for reader and writer modules without changing active encoding rules. (c9997fa)

UI/UX & Accessibility Fixes

  • Implemented strict coordinate math checks inside iOS WebView layouts, matching WebKit exception rules to prevent WKWebView crash exceptions during dynamic viewport changes. (f0eda5f)
  • Restored exact matching criteria based on renderer process IDs to ensure extension WebRequest listeners registered for sub-WebViews are completely uninstalled and cleaned up upon destruction. (58fe88b)
  • Patched Android tab visual bugs, resolving action button hover regressions on vertical layouts and re-enabling default empty tab creation when all active groups are collapsed. (aab72d9) (30942f5)
