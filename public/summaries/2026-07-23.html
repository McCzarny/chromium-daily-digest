Daily summary of Chromium changes for 2026-07-23 covering 797 total commits (621 relevant commits) spanning from first commit (728d0d5) to last commit (935e836).

Blink, Rendering & Web APIs

  • ⚠️[BREAKING CHANGE] Completely removed the non-standard AudioContext.playoutStats API and AudioPlayoutStats interface from WebAudio. (9a32d94)
  • ⚠️[BREAKING CHANGE] Removed the deprecated Topics API implementation and related interception logic from //content. (ca77f05)
  • Ignored keydown events where event.repeat is true in HTMLSelectElement and HTMLOptionElement default handlers to prevent key hold-down flashing. (6fa0d27)
  • Expanded Small String Optimization (SSO) with a 16-byte lock-free thread-local cache in AtomicStringTable to improve Speedometer 3 performance. (e6aea43)

Security, Privacy & Networking

  • Implemented Global Privacy Control (GPC) support by attaching the Sec-GPC header and exposing navigator.globalPrivacyControl. (9f6110d)
  • Restricted initial-empty-document paint-holding exemptions solely to browser-created tabs with unique opaque origins to mitigate origin spoofing. (ab2be7c)
  • Enforced initiator RenderFrameHost Connection-Allowlist policy checks within NetworkRequestManager::DownloadUrl for cross-origin FedCM requests. (a0a58fd)

UI, UX & Browser Features

  • Integrated horizontal and vertical tab strip transition scheduling and added drag-to-group metric logging for Android Vertical Tabs. (16d98b4)
  • Turned off glass frame styling on all browser windows when Battery Saver mode is active. (0640e71)
  • Re-integrated the Current Tab chip into the WebUI Omnibox popup UI and added custom HTML drag-and-drop preview images for selection text. (ada1eff)

Core Systems & Infrastructure

  • Cleaned up ~130 callsites to remove the legacy three-argument BASE_FEATURE macro in favor of the clean two-argument variant. (a410d46)
  • Refactored Ash and ChromeOS services (e.g., DeviceCommandsFactoryAsh, TokenHandleStoreFactory) to explicitly inject PrefService dependencies and avoid g_browser_process. (259a2e6)
  • Completed the multi-part refactoring campaign to entirely removeBrowser::profile() in favor of BrowserWindowInterface::GetProfile(). (601d560)
