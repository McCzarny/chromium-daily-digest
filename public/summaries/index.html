Chromium Digest: 2025-11-09

Overview

A total of 107 commits were made to the 'main' branch. After filtering, 22 relevant commits were analyzed. The day's changes span from commit (96c4358) to (cfa4cb1).

Performance

  • Introduces a new mechanism to defer navigations until a 'v8-optimization-verdict' is computed. This verdict, based on site familiarity from browsing history and Safe Browsing allowlists, determines whether to enable V8 optimizations for a given site, aiming to improve performance on familiar sites while potentially enhancing security on unfamiliar ones. The work is primarily located in chrome/browser/site_protection/. (9beb380)

Security

  • Routine updates to security metadata, including the Key Pinning (KP) list in transport_security_state_static.pins and the Certificate Transparency (CT) log list. (31bf870) (374814d)

Blink Engine

  • A significant update to the experimental 'HTML in Canvas' feature introduces a **breaking change** by removing the SetHitTestRegions API. Hit testing is now always-on and integrated with CSS transforms, which simplifies the API and enables support for anchor positioning. (3a9e6b2)
  • Improvements to Local Storage include the implementation of LocalStorageLevelDB::WriteMetadata(). This change provides a dedicated method for writing metadata such as last access and modification times to the LevelDB backend, which will be used to more accurately track storage map usage. (64b770d)
  • For WebNN development and testing, a **new** command-line switch, --webnn-ort-ep-device, has been added. This allows developers to force the selection of a specific ONNX Runtime execution provider device by name, vendor ID, and device ID. (98e6bd5)
  • The NavigatorPlugins partial interface has been refactored into an interface mixin to better align with the HTML specification. This is a code health improvement with no behavior change. (5a21b9a)

Developer Tools

  • The layer tree debugging utility in ui/compositor/debug_utils.cc has been improved to print the background-color of each layer, aiding in visual debugging. (905c682)

Bug Fixes

  • A crash in the omnibox (realbox) has been fixed by handling a null case for the expandedSearchboxShowVoiceSearch flag in ui/webui/resources/cr_components/searchbox/searchbox.ts. (f519a52)
  • A potential crash in single-process mode (--single-process) has been resolved by adding a null check for RendererMemoryCoordinatorPolicy, which does not exist in this mode. (7b5a317)
  • The logic for enabling image sharing has been updated in glic_enabling.cc to correctly check for managed accounts, ensuring policies are applied as expected. (903dd22)

Infrastructure

  • Various dependencies were rolled forward, including Web Platform Tests (WPT), ChromeOS LKGM, and iOS frameworks. (f3b75cf) (f2167f2) (b08ecd9) (eefc449)
  • Code health improvements include removing numerous superfluous includes of base/files/file_util.h and migrating some Android net tests from JUnit's Assert.assertThat to the more readable Truth framework. (502b6c6) (594a5ac)
