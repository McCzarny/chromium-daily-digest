Chromium Digest: 2025-11-09

Overview

A total of 107 commits were made to the 'main' branch. After filtering, 17 relevant commits were analyzed. The day's changes span from commit (96c4358) to (cfa4cb1).

Security

  • Introduces a 'Site Familiarity' check to defer navigation until a V8 optimization verdict is computed. This mechanism aims to enhance security by selectively enabling V8 optimizations only for sites that are considered familiar, based on browsing history and a safe browsing allowlist. This affects process selection logic in chrome/browser/site_protection. (9beb380)

Blink Engine

  • The HTML-in-canvas feature has been updated to support always-on hit testing via CSS transforms. This change removes the SetHitTestRegions API and enables better support for features like anchor positioning. (3a9e6b2)
  • Implements LocalStorageLevelDB::WriteMetadata() to record metadata for local storage, such as last modified and last access times. This improves how storage usage is tracked, particularly when a map is unloaded without any write operations. (64b770d)
  • Adds a new command-line switch, --webnn-ort-ep-device, for the WebNN API. This allows developers to force the selection of a specific ONNX Runtime execution provider device for development and testing purposes. (98e6bd5)
  • Refactors NavigatorPlugins from a partial interface to an interface mixin to align with the latest HTML specification. This is a code health improvement with no behavior change. (5a21b9a)

UI/UX

  • Fixes a potential crash in the omnibox (realbox) by adding a null check for the expandedSearchboxShowVoiceSearch feature flag in searchbox.ts. (f519a52)

Developer Tools

  • Enhances debugging capabilities by adding the layer's background-color to the layer tree dump output in ui/compositor/debug_utils.cc. (905c682)

Bug Fixes

  • Prevents a null pointer access crash in single-process mode (--single-process) by adding a check before using RendererMemoryCoordinatorPolicy. (7b5a317)
  • Corrects the logic for enabling image sharing features by ensuring that managed accounts are also checked in IsShareImageEnabledForProfile. (903dd22)

Infrastructure

  • A large-scale code health effort removed numerous superfluous includes of base/files/file_util.h across the codebase. (502b6c6)
  • Test frameworks were improved by migrating Android network tests to the Truth assertion library and adding a new ButtonTestApi::NotifyMouseClick helper for UI tests. (594a5ac) (0db0398)
  • Routine maintenance included updating WebNN test expectations, importing the latest Web Platform Tests (WPT), and updating translation string files. (cfa4cb1) (f3b75cf) (b2e14ec)
  • Minor code cleanups involved removing a Unicode 'Replacement Character' from a test file and deleting an outdated comment in html_tag_names.json5. (a1dadc2) (96c4358)
