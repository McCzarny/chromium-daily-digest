Overview

A total of 571 commits were recorded on 2026-06-22, with 408 identified as relevant for architectural, security, and functional updates. The day spanned from commit (53dcdec) to (f9744f8), focusing heavily on TabStripModel modernization and internal API refactorings.

Tab Strip & Window Management

  • ⚠️The TabStripModel API has been modernized, moving away from index-based manipulation toward object-oriented patterns. Several helper methods were removed, requiring embedders to interact directly with tabs::TabInterface. (d5e549a) (6f1965e)
  • ⚠️The tab drag system was migrated to a client-driven push model for bounds calculation, modifying the TabDragWindowAdapter interface to require a mandatory gfx::NativeView context. (7e297e1) (5519490)

Blink & Web Platform

  • Implemented a major overhaul of event propagation for CommandEvent, ToggleEvent, SubmitEvent, and InterestEvent, aligning them with the standard relatedTarget concept instead of internal trackers. (734bb1f)
    BlinkHTML
  • Removed the legacy 64k text-node splitting hack, allowing nodes to retain full length for improved performance. (e180fbb)
    Blink

Security & Network

  • Transitioned HTTP cache keys to string_view to eliminate heap allocations, while hardening the parser against malformed key integer rollovers. (0ed2703)
    NetworkPerformance
  • Implemented an enterprise proxying factory for HTTP header injection to ensure enterprise-injected headers take precedence over extension-modified headers. (8477053)
    NetworkEnterpriseSecurity

Infrastructure & Build

  • Extracted test targets for spellchecker and component_updater from the monolith chrome/test target into dedicated source sets. (d0a5270) (c8b4ef4)
