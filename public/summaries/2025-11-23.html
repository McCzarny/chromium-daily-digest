Chromium Digest: 2025-11-23

Overview

This digest covers 7 commits, analyzing changes across UI, networking, bug fixes, and testing infrastructure. No major breaking changes to stable public APIs were identified in this set of commits. Key updates include a UI change in Chrome for Android's app menu, a crash fix for the find bar, and enhancements to networking logs and test coverage.

UI/UX

  • The 'Pin tab' and 'Unpin tab' options have been removed from the main app menu in Chrome for Android's tabbed mode. This is a UI simplification and does not affect core embedder APIs. (a1dc4cc)
  • A new feature flag, SidePanelRelativeAlignment, has been added to chrome://flags for internal testing of new side panel alignment behaviors. (00599f1)

Bug Fixes

  • A potential crash on ChromeOS related to the Find Bar has been resolved. The fix prevents updates from non-tab WebContents from incorrectly interacting with the find bar controller, improving stability. (c169024)

Networking

  • To improve debuggability, new NetLog events have been added for proxy resolution. These logs will help identify when a proxy configuration is applied via an override rule. (b2e1560)

Testing & Infrastructure

  • The test suite has been enhanced with a new fuzz test targeting DOM scenarios known to be problematic for accessibility tree creation. (f147f3f)
  • The latest version of the Web Platform Tests (WPT) has been imported, ensuring test coverage aligns with the latest web standards. (4efd56a)
  • Minor code maintenance was performed, including the removal of obsolete TODO comments from the System Live Caption service header. (cb87ba9)
