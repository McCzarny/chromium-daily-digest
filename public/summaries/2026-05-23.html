Chromium Digest: 2026-05-23

Overview

This digest covers 55 commits, spanning from (b3ed2f7) to (33b8638). Key updates include security hardening for the Extensions API, stabilization of the RequestIsReloadNavigation feature, and ongoing refactoring in ChromeOS and the WebUI toolbar. No major breaking changes were identified in this set of commits.

Security & Privacy

  • A cross-origin secret leak vulnerability in the Extensions API has been fixed. The issue occurred when a compromised renderer sent two requests with the same ID, causing network callbacks to be misrouted. The IPC boundary is now hardened to detect this, terminate the offending renderer, and prevent data leakage. (048923c)
    SecurityExtensionsNetwork
  • A potential use-after-free vulnerability in Bluetooth socket listeners on macOS has been addressed. The fix ensures the listener object outlives any pending OS notifications, preventing crashes and potential security issues. (375096a)
    SecurityBluetoothmacOS
  • The experimental Compute Pressure API has had several recent changes reverted due to concerns about it creating an information leak. Active development on this part of the feature has been stopped. (4d118b5) (d235c83) (a8f43ad) (448f7cd)
    BlinkPrivacy
  • Password fields are now more reliably redacted in screenshots by using the TrackedElement infrastructure to identify and mask sensitive areas. (421393c)
    PrivacyPasswordManager

Blink & Web Platform

  • The RequestIsReloadNavigation feature has been enabled by default, graduating it to a stable web platform feature. This allows developers to more reliably detect if a navigation was initiated by a page reload. (33b8638)
    BlinkNavigation
  • The WebAudio AudioBufferSourceHandler has been refactored to better separate the linear-interpolation rendering path, improving code clarity and preparing for future features like negative playback rates. (7e40dfa)
    BlinkMedia
  • A bug fix in the GLES2 command decoder now properly handles EXT_texture_norm16 formats during ReadPixels calls, preventing potential rendering issues or crashes. (509449e)
    GPUWebGL

UI & UX

  • Development continues on the new WebUI-based toolbar. The security icon has been refactored into a native <button> for better accessibility, and an extensions container has been added in preparation for future integration. (b889e37) (1b16115)
    UIAccessibilityExtensions
  • Several UI refinements were made as part of the "Glow Up" project, including updating vector icons to a modern rounded style and fixing an animation bug with the page info icon. (f39794b) (a88841b)
    UIThemes
  • On Android, a bug was fixed where the toolbar snapshot would have the wrong theme color when initially positioned at the bottom of the screen. (bc43a71)
    UIAndroid

ChromeOS & Ash

  • A large-scale refactoring effort continues to decouple Ash components from the higher-level browser layer. Multiple components, including OobeUIConfig, GaiaScreenHandler, and OS Settings tests, have been migrated away from ash::ProfileHelper to use lower-level APIs like ash::BrowserContextHelper. (499757b) (2676051) (15e4855) (6b2787e)
    AshChromeOS
  • A change affecting task execution order during shutdown was relanded after a series of reverts. This change aims to catch errors earlier by running tasks before the shutdown process begins, which helps stabilize tests. (2af5fa9) (eef1101)
    AshBase

Infrastructure & Refactoring

  • The build system has been updated to support TypeScript v7 via a new use_typescript_go flag, enabling more modern frontend development practices for WebUI components. (1487cbd)
    BuildInfra
  • A codebase-wide migration is underway to replace calls to the deprecated base::SysInfo::AmountOfPhysicalMemory with AmountOfTotalPhysicalMemory. This ensures more accurate memory reporting across Blink, Compositor (cc), and ChromeOS components. (40d7c8c) (e8fde96) (eafe39a)
    BaseBlinkCompositor
  • The libvpx dependency has been updated, incorporating a fix that disables VP9D_DEMACROBLOCK for small video frame dimensions to prevent potential issues. (b33840f)
    MediaBuild
