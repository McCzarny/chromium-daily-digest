Chromium Digest: 2026-06-01

Overview

A total of 669 commits were analyzed for the 2026-06-01 period, with 505 deemed relevant for developers and embedders. The day focused on extensive security hardening, DevTools protocol evolution, and foundational infrastructure improvements. The first commit was (82cdf23) and the last was (8512f49).

Architectural Changes

  • ⚠️Refactored the DevTools protocol (CDP) dispatching logic, moving from asynchronous callbacks to a synchronous UberDispatcher model. Embedders implementing custom protocol handlers must update their implementations. (7a3b7b3)
    DevToolsBrowser

Security & Privacy

  • Implemented session-aware User SID verification and enforced untrusted process flags for Windows Named Mojo Pipe Servers to prevent local privilege escalation. (8941f7c)
    SecurityWindows
  • Patched multiple Use-After-Free (UAF) vulnerabilities in DesktopWindowTreeHostPlatform, Wayland window handling, and D3D12 video encoding accelerators. (7340932) (273ca3d) (5f73116)
    SecurityGPULinux

Blink & Web Platform

  • Expanded background-clip:text support to include stacking-context children, and enabled CSSZoomAnimation for continuous property interpolation. (a56f062) (0d90202)
    BlinkCSS
  • Systematic migration of raw pointer arithmetic and legacy code to base::span containers, particularly within Blink’s font-shaping and OpenType stacks. (83fc7c4)
    BlinkBaseFonts

Build & Infrastructure

  • Standardized Protobuf Well-Known Types (WKTs) with a new proto_wkt GN template and removed the deprecated is_chromeos_ash build argument. (ae6a841) (2c2d89e)
    BuildChromeOS
