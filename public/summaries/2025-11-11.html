Summary for 2025-11-11 on branch 'main'

Chromium Digest: 2025-11-11

Overview

A total of 300 commits were made to the 'main' branch. After filtering, 231 relevant commits were analyzed. The day's changes span from commit (6fd7a62) to (089db6c).

Blink Engine

  • The Supplementable class, a mechanism for attaching extra data to Blink objects, has been refactored from a hash-map to an array-based system. This major architectural change improves performance by replacing hash lookups with direct array access, resulting in reduced memory usage and a smaller binary size of ~182kB on Linux. (a0c6dba) (cdd031f)
  • Scroll events from keyboards now correctly respect the overscroll-behavior CSS property. When set to contain or none, scroll events will no longer bubble up to ancestor elements, aligning behavior with the specification. (22f6dfc)
  • Flexbox layout algorithm was improved to prevent overflow/underflow issues when distributing free space among items with non-zero flex factors, leading to more accurate and predictable layouts. (fbb8385)

Features

  • The DataSharingJoinOnly feature flag has been enabled by default. This change moves features related to data and tab group sharing towards a stable rollout, impacting services like TabGroupSyncService and CollaborationService. (bb742de)

UI/UX

  • ⚠️ BREAKING On Linux, the LinuxUiDelegate::ExportWindowHandle API was refactored to remove an overloaded variant. The remaining function signature was changed to return void and now always uses a callback, simplifying the interface for Wayland and X11 UI components. (089db6c)
  • Significant progress was made on the "Answers in Motion" (AIM) feature for Android's omnibox, including a new animated glif, restricting the file picker to PDFs, and general UI cleanup in the NavigationAttachments coordinator. (84784bf) (ca4b75e) (3cccb03) (7e5078e)
  • A new page action chip to toggle the toolbar height side panel was added, but later reverted due to test failures. (63fe4fd) (bbc9547)
  • The extension installation dialog on Android is being updated to display the permissions an extension requests before installation. (9ad9658)
  • Support for WindowOpenDisposition::OFF_THE_RECORD and NEW_WINDOW has been implemented in Android's asynchronous Navigate() function, improving behavior for opening incognito and new windows. (c5f16ca) (1c7c205) (a2cd70a)

Security

  • ⚠️ BREAKING The removal of the Partitioned Popins feature has begun with the deletion of its core IPC mojo interface. This change effectively disables the feature, which was only available as a developer trial, and is the first step in a series to remove all related code. (154f95a)
  • A large-scale effort continued to remove the --ignore-certificate-errors flag from numerous browser tests. This improves test suite security by requiring the use of properly configured HTTPS test servers instead of bypassing certificate validation. (55032fa) (8f56585) (ce0ad21) (59f6448) (798cbae) (56b1829)
  • Fixed a vulnerability in the 7-Zip reader where an excessively large reported header size could lead to an over-large memory allocation. The code now validates the header size against the file size before allocating memory. (f0e106c)

Bug Fixes

  • Fixed a dangling pointer in Linux file dialogs by adding a WeakPtrFactory to SelectFileDialogLinuxPortal, preventing crashes from DBus callbacks that could arrive after the listener was destroyed. (47ae931)
  • A bug in Progressive Web Apps (PWAs) that caused app launch failures due to corrupted icon metadata has been fixed. A database migration was added to clear the invalid icon information in PendingUpdateInfo, allowing affected PWAs to load correctly. (75b3835)
  • Corrected the logic for showing the global shortcuts dialog on Linux, preventing it from appearing unnecessarily on startup and ensuring it shows when a local shortcut is promoted to global. (7a8645f)
  • A recent change to exo::Buffer that removed GpuFenceHandle handling was reverted due to it being the suspected cause of a bug. (9dc0c83)

Infrastructure

  • The crabbyavif dependency was updated, incorporating 47 commits that improve the AVIF decoder, add better support for HEIC sequences, and include performance enhancements for Android. (b8cca30)
  • Various histogram definitions were modernized by migrating them from XML suffixes to token-based variants. This affects metrics for Service Workers, Prerender, and power usage on macOS. (28aecf0) (c7c62d6) (8be504e)
