Chromium Digest: 2026-07-05

Overview

This digest analyzes 3 commits in the Chromium repository. These updates implement ARIA form control extraction for iOS, add UMA telemetry for voice typing, and update localization bundles. No major breaking changes impacting stable public APIs or production embedder builds were found. First commit: (98ef65c), last commit: (8184370).

Web Semantics & Platform Parity

  • Implemented support for ARIA custom form control semantics on iOS during page content extraction to align with Blink. This maps ARIA roles (such as checkbox, switch, radio, searchbox, textbox, combobox) to their respective FormControlType and parses attributes like aria-required, aria-readonly, aria-placeholder, and aria-checked. It also populates the is_readonly field in the FormControlData proto wrapper on iOS. (8184370)
  • Added UMA metrics and tracking histograms for the Voice Typing Dictation onboarding and session controller flows. (98ef65c)

Internationalization & Localization

  • Synchronized XML Translation Bundle (.xtb) resource files across multiple languages (including Hindi, Lithuanian, Marathi, and Serbian) to align with defined source string configurations (.grd) on the main branch. (02e6b69)
