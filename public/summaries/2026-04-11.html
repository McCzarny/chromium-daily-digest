Chromium Digest: 2026-04-11

Overview

This digest covers 32 commits, from refs/heads/main@{#(1613178)} to refs/heads/main@{#(1613332)}. A major breaking change was identified in the content public API, where the PrePrefetchService::Create method signature was altered. Other notable changes include security hardening for CORS requests, UI enhancements for Tab Groups, and various platform-specific bug fixes for iOS and ChromeOS.

Content API

  • ⚠️ BREAKING The factory method PrePrefetchService::Create has a new signature, adding three parameters: initial_origin_hint, initial_javascript_enabled_hint, and initial_should_append_variations_header_hint. This change, part of a refactor to pre-calculate headers on the UI thread, requires embedders who directly use this public API to update their calls to provide the new arguments. (508c306)

Networking & Security

  • As a security enhancement, no-cors requests using forbidden HTTP methods like TRACE and TRACK are now blocked. Previously, this restriction only applied to the CONNECT method. (d513cd2)
  • A global buffer overflow in Cast Remoting's protocol handling was fixed by adding bounds checking when casting proto enum values to their C++ counterparts. (0ff4ac1)
  • Potential use-after-free instances in the Remoting protocol's authenticator logic have been resolved. (174583c)
  • The EnterpriseRealTimeUrlCheckNewUrl feature for enterprise Safe Browsing has been enabled by default. (2590745)

UI & UX

  • As part of the "Glow Up" project, a new set of alternate colors for Tab Groups has been added behind a feature flag. The new colors are now reflected in the tab strip, group editor bubble, and WebUI tab search menu. (3efb98d) (a69fa3e) (87013f2)
  • The Contextual Tasks feature now opens the side panel earlier during a Lens query to improve perceived responsiveness. Additionally, an "Undo" snackbar will now be shown when the bottom sheet is closed. (52fb6ea) (dfad241)

Blink & Rendering

  • A bug in Blink's GeometryMapper fast path has been fixed to correctly handle viewport mapping for out-of-process iframe (OOPIF) roots. This ensures remote transforms are applied correctly, preventing OOPIFs from returning incorrect local coordinates. The fix is gated by the FixVisualRectRemoteViewportTransform feature flag. (bb4aebf)

Internals & Memory

  • The base::AutoReset utility has been updated to use raw_ptr<T> instead of a raw C++ pointer, improving memory safety as part of the ongoing MiraclePtr migration. (c8d4183)
  • The mechanism for syncing accessibility preferences on ChromeOS has been updated to better handle conflicts between local OOBE settings and values from Sync. (5fc6fa7)

Platform Specific

  • Several iOS-specific build issues and crashes have been resolved, including a crash in extension processes related to PartitionAlloc, missing method implementations in ContentWebState, and a build error in WebNN CoreML. (9c0c4da) (eb775c6) (b3bb527)
  • On ChromeOS, the Fjord OOBE (Out-of-Box Experience) will now display an error screen if the selected device image fails to download. (ef90817)

Metrics & Telemetry

  • A new suite of unified metrics for voice search has been introduced to centralize logging logic, which was previously fragmented across different surfaces. (4ac984b)
  • An expired histogram, Extensions.ErrorCodeFromCrxOpen, which had no recording sites, has been removed. (b9bcdde)
