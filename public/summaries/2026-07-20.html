Overview

A total of 557 commits were integrated into the Chromium main branch on July 20, 2026, with 428 deemed relevant to developer and infrastructure workflows. The range spans from commit (eebba89) to (6c1b3de).

Web Platform & Blink

  • ⚠️The document.browsingTopics() API has been removed, resulting in a DOMException when accessed, marking a breaking change for developers relying on the Topics API. (7628f2a)
    BlinkPrivacy
  • Updated CSS serialization for background-size and mask-size to include implicit auto heights, aligning with the latest CSSWG drafts. (5333161)
    BlinkCSS
  • The alpha() color function is now enabled by default in the CSS engine. (6a7e22a)
    BlinkCSS

Infrastructure & Build

  • Ongoing modularization efforts continue, with significant extraction of sub-targets for browser UI components and test support libraries to resolve circular dependencies and improve build times. (dc08bf0) (e14cba4) (c48cc6e)
    BuildInfraTesting
  • Migration of Android javatests to host-side unit tests using BaseRobolectricTestRunner continues, enhancing test speed and reliability. (60b929f) (cc8f7e8)
    AndroidTesting

Security & Stability

  • Enabled Win32k lockdown policy for the OnDeviceModelExecution process to harden renderer isolation on Windows. (be73280)
    SecurityWindows
  • Resolved critical Use-After-Free (UAF) vulnerabilities on macOS by switching modal sheet and bubble widget destruction to asynchronous task posting. (d4c2f03) (3c03058)
    macOSSecurity

Browser Features & AI Integration

    AIBrowserEnterprise
  • Improved WebGPU status checks by requiring explicit comparison against wgpu::Status::Success, moving away from implicit boolean conversion. (76f69ec)
